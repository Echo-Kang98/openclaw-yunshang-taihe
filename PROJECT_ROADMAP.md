## 三、分阶段推进方案

### 阶段一：数据基础（第1-4周）

**目标**：打通数据层，让 AI 有真实数据可用

| 任务 | 负责人（建议） | 内容 |
|------|---------------|------|
| Tushare API 接入 | 后端开发 | 注册Tushare账号，获取行情数据（SC原油、Brent） |
| 外汇数据 | 后端开发 | 接入 USD/CNY、EUR/CNY 实时汇率 |
| Notion 数据库设计 | 全栈/产品 | 设计订单、合同、库存、财务等数据表 |
| Notion 数据库创建 | 全栈/产品 | 在 Notion 创建业务数据库 |
| Skill 基类改造 | 后端开发 | BaseSkill.fetch_data() 改为真实 API 调用 |

**交付物**：
- 行情数据 API 服务
- Notion 业务数据库（空壳）
- 3个 Skill 的真实 API 版本

---

### 阶段二：Skill 深化（第5-10周）

**目标**：让核心业务 Skill 可用，支撑实际业务

**交付物**：
- 8-10个可用的真实 Skill
