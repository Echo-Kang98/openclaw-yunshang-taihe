Gateway 是什么？

养殖龙虾的第一步，是买虾苗。OpenClaw 的 Gateway 就是这袋"虾苗"——它是整个系统的核心，没有它，后面一切都不存在。

🏪
Gateway
OpenClaw 的核心服务，像养殖基地的总控室
🌐
7×24 运行
像永不休息的管家，时刻待命
📡
API 接口
所有 Agent 通过它通信协作

🦐 放虾苗
配置第一个 Agent

池塘挖好了！现在放入第一批虾苗——你的第一个 Agent。

🤖
Agent

Memory

⏰ 自动投喂机
设置定时任务 Cron

不想每天手动喂虾？设置自动投喂机——Cron 定时任务，让龙虾每天自动吃饭！

传统方式Cron 定时任务
每天手动搜集新闻每天 8 点自动搜集
记着发日报定时自动发送到飞书
容易忘记永不遗漏
# 每天早上 8 点自动喂虾（发简报） openclaw cron add \ --name "每日简报" \ --schedule "0 8 * * *" \ --task "搜集新闻，生成摘要" \ --channel feishu
