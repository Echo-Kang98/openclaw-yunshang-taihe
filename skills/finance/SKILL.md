# 财务经营部综合技能

## 概述

财务经营部负责云上泰和集团的财务经营分析工作，下辖四个核心技能：

| 子技能 | 说明 |
|--------|------|
| 财务报表生成 | 月报/季报/年报自动生成 |
| 发票与结算管理 | 贸易相关发票处理与结算 |
| 资金调度管理 | 资金流向分析与调度建议 |
| 预算执行监控 | 预算执行进度跟踪与预警 |

## 触发条件

当用户提到以下关键词时激活：
- "财务报表"、"财务报告"、"月报"、"季报"、"年报"
- "发票"、"结算"、"付款"
- "资金"、"预算"、"执行率"

## 子技能详细说明

### 财务报表生成

当用户请求生成财务报表时：

1. **确定报表类型**：月报 / 季报 / 年报
2. **收集数据**：调用财务数据接口（如 Tushare 或企业 ERP）
3. **生成报告**：按模板填充数据
4. **输出**：Markdown 或 Excel 格式报表

**数据来源**（预留 API）：
- Tushare 财务数据接口
- 企业 ERP 系统 API

### 发票与结算管理

1. **录入发票信息**：发票号、金额、税率、交易方
2. **核对单证一致性**：确保发票与合同/提单匹配
3. **生成结算单**：计算应付/应收金额
4. **状态跟踪**：已开票/已付款/已核销

### 资金调度管理

1. **分析资金流向**：监控账户余额变化
2. **识别闲置资金**：找出可调度闲置资金
3. **调度建议**：提出资金调拨方案
4. **风险提示**：流动性风险预警

### 预算执行监控

1. **对比预算 vs 实际**：计算执行率
2. **进度预警**：执行率偏离阈值时告警
3. **差异分析**：找出超支/节余原因
4. **调整建议**：预算调整方案

## 回复模板

- 报表生成：`📊 正在生成{类型}报表，预计{time}完成...`
- 发票处理：`🧾 发票{编号}已{状态}，金额{金额}`
- 资金调度：`💵 当前闲置资金{金额}，建议调拨{方案}`
- 预算预警：`⚠️ {部门}预算执行率{占比}，{超支/节余}{金额}`
