# 期现交易部综合技能

## 概述

期现交易部负责期货与现货市场的套利、对冲及风险管理工作，下辖四个核心技能：

| 子技能 | 说明 |
|--------|------|
| 期货交易执行 | 支持 SC/Brent/WTI/Dubai 品种下单 |
| 套利策略分析 | 跨期/跨品种套利机会识别 |
| 期现对冲策略 | 现货与期货对冲方案设计 |
| 期货账户风险管理 | 持仓监控、保证金预警、清算管理 |

## 触发条件

当用户提到以下关键词时激活：
- "期货"、"合约"、"开仓"、"平仓"、"持仓"
- "套利"、"对冲"、"价差"
- "保证金"、"强平"、"追保"
- "SC"、"Brent"、"WTI"、"Dubai"

## 子技能详细说明

### 期货交易执行

**支持品种**：
- SC（上海原油）
- Brent（布伦特原油）
- WTI（美国原油）
- Dubai（迪拜原油）

**操作流程**：
1. **确认交易指令**：合约代码、方向（买/卖）、手数、价格
2. **风险预检**：检查保证金是否足够、持仓是否超限
3. **执行下单**：调用期货公司 API（或模拟数据）
4. **确认回报**：记录成交价格、时间、手数
5. **更新持仓**：调整持仓记录

**注意**：当前阶段使用模拟数据，暂不接实盘。

### 套利策略分析

1. **跨期套利**：同品种不同合约价差分析
2. **跨品种套利**：相关品种（如 SC-Brent）价差分析
3. **统计套利**：均值回归模型识别异常价差
4. **策略输出**：套利方向、入场点位、止盈止损

### 期现对冲策略

1. **对冲需求识别**：企业现货敞口分析
2. **合约选择**：主力合约 vs 特定合约
3. **对冲比例**：Delta 中性或套保比率计算
4. **动态调整**：根据现货变化调整期货仓位
5. **效果评估**：对冲效率 vs 成本分析

### 期货账户风险管理

1. **持仓监控**：实时盯市、浮动盈亏
2. **保证金预警**：风险度 > 80% 预警，> 100% 强平
3. **清算管理**：到期合约现金结算处理
4. **日报生成**：每日持仓、盈亏、资金使用报告
