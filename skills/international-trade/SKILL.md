# 国际交易部综合技能

## 概述

国际交易部负责跨境大宗商品贸易全流程管理，下辖四个核心技能：

| 子技能 | 说明 |
|--------|------|
| 国际供应商关系管理 | 供应商开发、评估、维护 |
| 国际贸易合同管理 | 合同起草、审阅、履约跟踪 |
| 国际贸易执行 | 报关、物流、结算全流程 |
| 外汇对冲策略管理 | 汇率风险对冲方案设计与执行 |

## 触发条件

当用户提到以下关键词时激活：
- "国际"、"进口"、"出口"、"跨国"
- "供应商"、"合同"、"贸易条款"
- "FOB"、"CIF"、" CFR"、"EXW"
- "外汇"、"汇率"、"对冲"、"USD"、"EUR"

## 子技能详细说明

### 国际供应商关系管理

1. **供应商档案**：建立全球供应商数据库
2. **资质审核**：信用评级、交货能力、质量体系
3. **绩效评估**：按时交付率、质量合格率、价格竞争力
4. **关系维护**：定期沟通、年度评审、战略合作

### 国际贸易合同管理

1. **合同起草**：贸易条款（FOB/CIF）、数量、价格、品质标准
2. **风险条款**：不可抗力、违约金、争议解决
3. **审阅流程**：法务审核 → 业务确认 → 签章存档
4. **履约跟踪**：付款节点、发货节点、交货节点

### 国际贸易执行

1. **信用证（LC）开立**：向开证行申请，跟踪开证进度
2. **国际物流**：订舱、装运、到港、清关
3. **单证管理**：提单(B/L)、产地证(CO)、商检(CIQ)
4. **结算付款**：T/T、D/P、L/C 结算方式处理

### 外汇对冲策略管理

1. **汇率风险识别**：敞口币种、金额、期限
2. **对冲工具选择**：远期、期权、掉期、货币互换
3. **策略设计**：部分对冲 vs 全额对冲，动态调整机制
4. **执行与评估**：对冲成本 vs 风险敞口效益分析

## 回复模板

- 供应商评估：`🏭 供应商{名称}评分{分数}，{优势}/{劣势}`
- 合同状态：`📄 合同{编号}状态：{节点}，{时间}`
- 物流更新：`🚢 货物已{节点}，预计{时间}到港`
- 对冲建议：`💱 {币种}汇率风险{金额}，建议{对冲工具}，成本{成本}`
