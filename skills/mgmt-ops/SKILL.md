# 卓越运营部综合技能

## 概述

卓越运营部负责集团运营效率提升与合规管理工作，下辖四个核心技能：

| 子技能 | 说明 |
|--------|------|
| KPI 监控看板 | 核心 KPI 实时监控与可视化 |
| 合规检查审计 | 贸易合规、监管合规定期审查 |
| 效能分析报告 | 部门/业务线效能分析 |
| 流程优化诊断 | 业务流程瓶颈识别与优化建议 |

## 触发条件

当用户提到以下关键词时激活：
- "KPI"、"指标"、"达成率"
- "合规"、"审计"、"检查"
- "效能"、"效率"、"优化"
- "流程"、"瓶颈"、"诊断"

## 子技能详细说明

### KPI 监控看板

1. **指标定义**：核心 KPI（贸易量、利润、周转率、回款天数）
2. **数据采集**：从 ERP、TMS、财务系统自动抽取
3. **实时更新**：每日/每周刷新看板数据
4. **预警机制**：指标偏离目标 > ±10% 自动预警
5. **可视化输出**：表格 + 图表（柱状/折线/饼图）

### 合规检查审计

1. **合规框架**：行业法规（海关、税务、外汇）、内部制度
2. **检查清单**：年度/季度合规检查清单
3. **问题整改**：发现不合规项 → 整改要求 → 跟踪闭环
4. **审计报告**：合规审计结论 + 风险评级

### 效能分析报告

1. **数据归集**：各部门/业务线经营数据
2. **横向对比**：各部门效率指标对比
3. **趋势分析**：同比/环比变化
4. **问题定位**：低效能部门/环节识别
5. **改进建议**：效能提升具体措施

### 流程优化诊断

1. **现状梳理**：核心业务流程图绘制
2. **瓶颈识别**：耗时最长/成本最高/出错率最高节点
3. **根因分析**：5Why 分析法
4. **优化方案**：流程简化/自动化/并行化建议
5. **收益测算**：优化后效率提升/成本节省预估

## 回复模板

- KPI 预警：`📊 {指标}当前{值}，偏离目标{偏离度}，{预警等级}`
- 合规结论：`✅ 合规检查通过 / ⚠️ 发现{数量}项不符合项`
- 效能报告：`📈 {部门}效能得分{分数}，{同比变化}`
- 优化建议：`🔧 {流程}优化：{当前问题} → {优化方案}，预计{收益}`
