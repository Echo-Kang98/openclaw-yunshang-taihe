# 供应链运营部综合技能

## 概述

供应链运营部负责集团供应链全链条优化管理，下辖四个核心技能：

| 子技能 | 说明 |
|--------|------|
| 仓储管理 | 油品/大宗商品仓储运营与安全管理 |
| 供应链管理 | 供应链全链条协调与优化 |
| 库存优化 | 库存水平分析与优化建议 |
| 物流成本分析 | 物流费用分析与降本方案 |

## 触发条件

当用户提到以下关键词时激活：
- "仓储"、"库存"、"入库"、"出库"
- "供应链"、"物流"、"运输"
- "库存周转"、"呆滞"、"盘点"
- "运费"、"物流成本"、"降本"

## 子技能详细说明

### 仓储管理

1. **罐区管理**：油罐液位、温度、压力实时监控
2. **出入库操作**：入库登记、出库计量、损耗管理
3. **安全管理**：防溢罐、气体检测、消防设施检查
4. **库存台账**：实时库存台账与盘点对账
5. **损耗分析**：途耗、储存损耗量分析与控制

### 供应链管理

1. **供应链可视化**：从采购到配送全链条跟踪
2. **供应商协同**：VMI（供应商管理库存）机制
3. **风险预警**：供应商断供、物流延误预警
4. **协调会议**：供应链各环节联席协调机制

### 库存优化

1. **库存结构分析**：周转库存、安全库存、呆滞库存分类
2. **库存周转率**：周转天数、周转次数分析
3. **补货策略**：安全库存计算、最佳补货点确定
4. **呆滞处理**：超期库存识别与处理方案
5. **优化建议**：ABC 分类、JIT 库存模式建议

### 物流成本分析

1. **运费结构**：按品类的运费组成分析
2. **物流对标**：与行业平均/最优对比
3. **路径优化**：运距最短/成本最低路径规划
4. **降本空间**：批量运输、拼车、仓储整合建议
5. **月度分析**：物流成本月报与趋势分析

## 回复模板

- 库存预警：`📦 {品名}库存{数量}，可维持{天数}天，{补货建议}`
- 供应链风险：`⚠️ 供应商{名称}断供风险，{建议措施}`
- 物流降本：`🚛 物流成本{金额}，优化空间{空间}，建议{方案}`
- 仓储状态：`🏭 罐区{名称}液位{液位}%，{状态}`
