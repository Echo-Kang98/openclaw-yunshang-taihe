# 交易执行部综合技能

## 概述

交易执行部负责贸易订单的全流程执行与协调，下辖四个核心技能：

| 子技能 | 说明 |
|--------|------|
| 订单执行管理 | 贸易订单全生命周期管理 |
| 单据与文件管理 | 贸易单证制作、审核、归档 |
| 船务与航次管理 | 租船、订舱、航次跟踪 |
| 物流协调管理 | 物流各环节协调与异常处理 |

## 触发条件

当用户提到以下关键词时激活：
- "订单"、"执行"、"贸易执行"
- "单据"、"提单"、"B/L"、"发票"
- "船务"、"航次"、"租船"、"运费"
- "物流"、"协调"、"到港"、"清关"

## 子技能详细说明

### 订单执行管理

1. **订单录入**：合同信息 → 系统建档
2. **执行跟踪**：节点管理（确认→备货→发货→到港→结算）
3. **异常处理**：交货延迟、品质异议、数量短缺
4. **状态更新**：实时订单状态可视化
5. **关闭归档**：订单完成 → 归档 → 数据统计

### 单据与文件管理

1. **单据清单**：提单(B/L)、发票(INV)、装箱单(PL)、原产地证(CO)、商检证书(CIQ)
2. **单证审核**：单证一致性核对（货物、数量、价格）
3. **制作流程**：标准格式 → 审核 → 签发
4. **归档管理**：电子归档 + 纸质存档
5. **时效管理**：交单期、信用证有效期跟踪

### 船务与航次管理

1. **租船方式**：程租、期租、光租选择
2. **航线规划**：装港→卸港路径、运费报价
3. **航次跟踪**：装货 → 航行 → 卸货全程监控
4. **港口协调**：锚地排队、泊位安排、卸货进度
5. **单船效益**：航次收益 vs 成本分析

### 物流协调管理

1. **多式联运**：海运+陆运+仓储衔接
2. **时效管理**：各环节时效标准 vs 实际对比
3. **异常协调**：滞箱、滞港、货物损坏处理
4. **信息同步**：各参与方信息实时共享
5. **成本控制**：物流成本最优化方案

## 回复模板

- 订单状态：`📋 订单{编号}：{状态}，{节点进度}`
- 单证完成：`🧾 单证{类型}已签发，{份数}份，交单期{日期}`
- 航次更新：`🚢 {船名}航次{编号}：{装港}→{卸港}，预计{到港日期}`
- 物流异常：`⚠️ {环节}异常：{描述}，处理中：{方案}`
