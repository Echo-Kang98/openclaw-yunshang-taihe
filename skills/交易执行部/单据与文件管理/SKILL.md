# 单据与文件管理

## 功能定位

管理石油/大宗商品国际贸易全流程所需单据，确保"单证一致、单货一致"，保障贸易安全。

## 核心单据类型

| 单据类型 | 英文缩写 | 用途 |
|----------|----------|------|
| 提单 | B/L | 物权凭证、运输证明 |
| 商业发票 | INV | 结算凭证 |
| 装箱单 | P/L | 包装明细 |
| 原产地证 | CO | 关税优惠、海关申报 |
| 商检证书 | CIQ | 品质/数量检验 |

## 操作流程

### 1. 单据制作

**触发**：贸易合同签订后自动创建单据任务

```
1. 收集基础信息：合同号、货物品名、数量、装港卸港
2. 选择单据模板：标准格式或买方指定格式
3. 填制单据内容：确保与合同、信用证条款一致
4. 内部审核：单人复核 → 确认无误
5. 签发存档：电子版归档、纸质版存档
```

### 2. 单证一致性审核

**触发**：收到客户提供的单据草稿时

**审核要点**：
- 货物描述与合同一致
- 数量与提单一致
- 价格与信用证/合同一致
- 唛头与实际包装一致
- 装港卸港与合同一致

**输出**：审核报告（通过/修改意见）

### 3. 单据交付

**触发**：货物装运后

**操作**：
1. 确认正本单据份数
2. 选择交付方式（快递/自取）
3. 跟踪单据寄送状态
4. 确认客户签收

## 常见问题处理

| 问题 | 处理方式 |
|------|----------|
| 单据内容与合同不符 | 退回修改，重新审核 |
| 交单期临近 | 加急办理，通知客户 |
| 单据丢失 | 登报声明，补办副本 |
| 信用证修改 | 评估影响，沟通开证行 |

## 回复模板

- 单据创建：`📄 单据任务已创建：{单据类型} × {数量}份`
- 审核通过：`✅ {单据类型}审核通过，内容无误`
- 审核驳回：`❌ {单据类型}需修改：{问题描述}`
- 单据寄送：`📮 {单据类型}已寄出，预计{日期}到达`
