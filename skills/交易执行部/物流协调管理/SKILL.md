# 物流协调管理

## 功能定位

协调国际贸易物流全链条，确保货物按时交付、成本最优、异常及时处理。

## 核心业务场景

### 1. 物流计划制定

**触发**：订单确认或合同签订后

**流程**：
```
1. 分析货物特性：品类、数量、时效要求
2. 选择运输方式：海运/陆运/铁路/多式联运
3. 规划运输路线：装港→路径→卸港→目的地
4. 确定时效节点：各环节标准时效
5. 预估物流成本：运费+保险+清关+仓储
```

### 2. 多式联运协调

**场景**：门到门全程物流

**协调要点**：
- 海运段：船期、舱位、装港
- 陆运段：拖车、报关、转运
- 仓储段：卸港仓储、入库
- 最后一公里：配送安排

### 3. 时效管理

**监控指标**：
- 各环节标准时效 vs 实际时效
- 延误预警（>2小时触发）
- 整体交付及时率

**预警机制**：
| 预警等级 | 条件 | 处理 |
|----------|------|------|
| 🟡 黄色 | 延误 2-6 小时 | 加速后续环节 |
| 🟠 橙色 | 延误 6-24 小时 | 启动备选方案 |
| 🔴 红色 | 延误 > 24 小时 | 升级处理，通知客户 |

### 4. 异常处理

**常见异常类型**：
- 滞箱/滞港：箱源/泊位紧张导致额外费用
- 海关查验：货物被海关抽检，延误清关
- 货物损坏：运输途中货损
- 天气影响：台风、大雾导致船期延误

**处理流程**：
```
1. 异常识别 → 记录发生时间、地点、原因
2. 影响评估 → 对交付时效、成本影响
3. 方案制定 → 备选方案（转运、改港、加急清关）
4. 执行跟踪 → 实时更新处理进度
5. 闭环确认 → 异常处理完成，归档记录
```

## 物流成本分析

**成本构成**：
- 海运费：基本运费 + 附加费（BAF、PSS）
- 陆运费：拖车费、装卸费
- 清关费：报关费、查验费
- 仓储费：港口仓储、目的地仓储
- 保险费：货运险

**优化方向**：
- 批量运输降低单位成本
- 拼箱运输适合小批量
- 淡季囤货降低运费成本

## 回复模板

- 计划制定：`🚛 物流方案已规划：{路线}，时效{天数}天，成本{金额}`
- 时效预警：`⏰ {环节}延误{小时}小时，当前处理{措施}`
- 异常处理：`⚠️ {异常类型}：{描述}，已启动{方案}，预计{时间}解决`
- 成本优化：`💰 物流成本{原金额} → {优化后金额}，节省{节省金额}`
