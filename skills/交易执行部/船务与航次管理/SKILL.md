# 船务与航次管理

## 功能定位

管理油品/大宗商品海运业务，包括租船、订舱、航次跟踪、运费结算，确保货物按时装运交付。

## 核心业务

### 1. 租船与订舱

**触发**：贸易合同签订，需要安排海运时

**租船方式选择**：

| 类型 | 特点 | 适用场景 |
|------|------|----------|
| 程租（Voyage） | 按航次计费 | 大宗商品整船运输 |
| 期租（TC） | 按时间计费 | 长期稳定货源 |
| 光租（Bareboat） | 船舶运营权出租 | 长期自营 |

**订舱流程**：
```
1. 确认船期：与船东/货代确认可用舱位
2. 提交订舱委托：品名、数量、装港、卸港、受载期
3. 舱位确认：船东确认舱位，发放订舱确认书（S/O）
4. 预配载：确认货物配载位置
5. 装货准备：准备装货文件、确保货物就位
```

### 2. 航次跟踪

**航次阶段**：

| 阶段 | 关键节点 | 监控内容 |
|------|----------|----------|
| 装货前 | 受载期→锚地→泊位 | 港口排队时间 |
| 装货中 | 开始装货→装货完毕 | 装货进度、数量 |
| 航行中 | 离港→航行→预到 | 预计到港时间 |
| 卸货中 | 到港→泊位→卸货 | 卸货进度 |
| 卸货后 | 完毕→离港 | 提单签发 |

**跟踪报告**：
- 每日更新航次状态
- 到港前24小时确认卸港安排
- 异常情况（天气、机械故障）即时通报

### 3. 运费结算

**运费构成**：
- 基本运费（Ocean Freight）
- 燃油附加费（BAF：Bunker Adjustment Factor）
- 旺季附加费（PSS：Peak Season Surcharge）
- 港口拥挤费（PTF：Port Congestion Surcharge）
- ，超卸费（L/I：Lighterage）

**结算方式**：
- 预付（Freight Prepaid）：发货方支付
- 到付（Freight Collect）：收货方支付

**对账流程**：
```
1. 核对运价：确认与合同/订舱一致
2. 核对数量：实际装货量 vs 预配量
3. 确认附加费：BAF/PSS 等是否如实结算
4. 开具发票：船东出具运费发票
5. 付款赎单：预付情况下完成付款
```

## 回复模板

- 订舱确认：`🚢 舱位已确认：船名{船名}，航次{航次}，装港{装港}→卸港{卸港}`
- 装货完成：`📦 装货完毕：实际装货量{数量}{单位}，离港时间{时间}`
- 航次更新：`🚢 {船名}当前状态：{状态}，预计{时间}{地点}`
- 运费结算：`💵 运费确认：{金额}，结算方式{方式}，发票{编号}`
