# 仓储管理

## 功能定位

管理油品/大宗商品仓储运营，确保库存准确、仓储安全、损耗可控。

## 核心业务

### 1. 罐区管理

**罐区监控指标**：
- 液位：当前储量 vs 罐容
- 温度：油品温度监控（影响体积计算）
- 压力：压力容器安全监控
- 含水：油品含水率监控

**罐容管理**：
- 罐容利用率 = 在用容积 / 总罐容
- 罐容预警：利用率 > 85% 预警新库容需求
- 周转计划：入库与出库节奏协调

### 2. 出入库操作

**入库流程**：
```
1. 入库预报：货主/船东提前通知
2. 罐号确认：分配存储罐位
3. 计量交接：流量计或人工计量
4. 品质验收：取样化验
5. 卸货入库：泵送入库
6. 计量确认：入库量确认
7. 单据签收：入库单签发
```

**出库流程**：
```
1. 出库指令：客户/销售发起
2. 罐号确认：确认出库罐
3. 计量出库：流量计计量
4. 品质确认：与客户确认品质
5. 装车/装船：出库装运
6. 计量确认：出库量确认
7. 单据签收：出库单签发
```

### 3. 安全管理

**日常安全检查**：
- 防溢罐：高液位报警
- 气体检测：可燃气体浓度
- 消防设施：灭火器、泡沫系统、消防栓
- 避雷防静电：接地系统检查

**应急预案**：
- 溢油应急：围油栏、吸油材料
- 泄漏应急：堵漏工具、应急池
- 火灾应急：消防队伍联系、疏散路线

### 4. 库存台账

**台账内容**：
- 罐号、货主、品名
- 库存量（体积/重量）
- 品质指标
- 入库时间、货龄

**盘点对账**：
- 日常台账 vs 计量数据核对
- 月度实物盘点
- 损耗账务处理

### 5. 损耗分析

**损耗类型**：
- 运输损耗（途耗）：运输过程中的洒漏、蒸发
- 储存损耗：呼吸损耗、挂壁损耗
- 作业损耗：装卸过程中的残留

**损耗控制**：
- 途耗率控制：装卸操作规范
- 储存损耗：浮顶罐选用、喷淋降温
- 残油回收：卸净后残油回收

## 回复模板

- 罐区状态：`🏗 罐区{罐号}：液位{液位}%，温度{温度}℃，状态{正常/预警}`
- 入库完成：`📥 {货主}已入库{品名}{数量}，罐号{罐号}，入库时间{时间}`
- 出库完成：`📤 {货主}已出库{品名}{数量}，罐号{罐号}，出库时间{时间}`
- 安全预警：`⚠️ {类型}预警：{描述}，已{处理措施}`
- 损耗报告：`📉 损耗分析：运输损耗{途耗率}%，储存损耗{存耗率}%，总损耗{总量}`
