# 供应链管理

## 功能定位

协调供应链全链条运营，实现供应链可视化、风险可控、协同高效。

## 核心业务

### 1. 供应链可视化

**可视范围**：
- 供应商端：原料采购进度
- 在途：运输状态
- 仓储端：库存水平
- 客户端：交付状态

**实现方式**：
- 供应链系统集成（ERP/TMS/WMS）
- 实时数据推送
- 异常自动预警

### 2. 供应商协同（VMI）

**VMI 机制**：
- 供应商管理库存（Vendor Managed Inventory）
- 供应商监控客户库存，自主补货
- 降低双方库存成本

**协同要点**：
- 信息共享：库存数据实时同步
- 补货规则：安全库存+补货点+补货量
- 结算机制：VMI 消耗结算

### 3. 风险预警

**风险类型**：

| 风险类型 | 预警指标 | 影响 |
|----------|----------|------|
| 供应商断供 | 供应商交货延迟 > 3天 | 生产中断 |
| 物流延误 | 运输时效延误 > 1天 | 客户交付延迟 |
| 库存断货 | 库存 < 安全库存 | 销售损失 |
| 质量异常 | 批次合格率 < 98% | 客户索赔 |

**预警处理**：
```
1. 预警触发 → 系统自动通知
2. 影响评估 → 评估对交付/成本的影响
3. 方案制定 → 备选供应商/加急物流/调拨库存
4. 执行跟踪 → 方案执行进度跟踪
5. 预警解除 → 风险消除，关闭预警
```

### 4. 协调会议机制

**例会类型**：
- 日站会：供应链运营每日碰头（15分钟）
- 周例会：供应链风险与计划周会
- 月度评审：供应商绩效月度评审
- 季度战略：供应链战略季度复盘

**会议内容**：
- 各环节运行回顾
- 风险与问题升级
- 下阶段计划协调
- 改善举措追踪

## 供应链关键指标

| 指标 | 定义 | 目标 |
|------|------|------|
| 订单履约率 | 按时交付订单 / 总订单 | > 98% |
| 库存周转率 | 年销售量 / 平均库存 | 行业领先 |
| 断货次数 | 库存断货发生次数 | < 2次/月 |
| 供应商准时率 | 准时到货供应商 / 总供应商 | > 95% |

## 回复模板

- 供应链状态：`🔗 供应链整体状态：{正常/预警}，今日履约率{履约率}%`
- 风险预警：`⚠️ {风险类型}：{描述}，影响范围{范围}，建议{措施}`
- VMI更新：`📦 {供应商}VMI补货：{品名}{数量}，预计{时间}到货`
- 会议通知：`📅 供应链{例会类型}将于{时间}召开，请准备{议题}`
