# 库存优化管理

## 功能定位

通过库存结构分析、安全库存优化、呆滞库存处理，提升库存周转率，降低库存持有成本。

## 核心业务

### 1. 库存结构分析

**库存分类**：

| 类型 | 特征 | 管理策略 |
|------|------|----------|
| 周转库存 | 正常销售/生产用 | 确保供应 |
| 安全库存 | 缓冲应急 | 合理水位 |
| 战略库存 | 应对重大风险 | 长期储备 |
| 呆滞库存 | 无需求或滞销 | 促销/处理 |

**ABC 分类**：

| 分类 | 占比 | 管控重点 |
|------|------|----------|
| A类（top 70%） | 价值高，周转快 | 精确管理 |
| B类（middle 20%） | 价值中等 | 常规管理 |
| C类（bottom 10%） | 价值低 | 简化管理 |

### 2. 安全库存计算

**计算方法**：

```
安全库存 = Z × σ × √(LT)

其中：
- Z = 服务水平对应的安全系数（95%: 1.65, 99%: 2.33）
- σ = 需求标准差
- LT = 供应提前期（天）
```

**影响因素**：
- 需求波动：需求变化越大，安全库存越高
- 供应波动：交货期波动越大，安全库存越高
- 服务水平要求：服务水平要求越高，安全库存越高

### 3. 补货策略

**定量订货（EOQ）**：
- 补货点 = 安全库存 + 日均需求 × 补货周期
- 经济批量 = √(2 × 年需求量 × 订货成本 / 单位持有成本)

**定期订货**：
- 补货周期：每周/每两周/每月固定补货
- 补货量 = 目标库存 - 当前库存 - 在途库存

**JIT 补货**：
- 按需采购/生产
- 最小化库存
- 适用于需求稳定、供应商响应快

### 4. 呆滞库存处理

**呆滞识别标准**：
- 超期库龄：入库 > 6个月无动态
- 无需求记录：连续 > 3个月无出库
- 品质问题：不合格品无法使用

**处理方案**：
- 内部调拨：其他部门/项目有需求
- 降价促销：折价销售回笼资金
- 退货供应商：与供应商协商退货
- 报废处理：无法使用的物资申请报废

## 库存分析指标

| 指标 | 计算 | 目标 |
|------|------|------|
| 库存周转率 | 年销售量 / 平均库存 | 行业领先 |
| 库存周转天数 | 365 / 周转率 | < 30天 |
| 呆滞库存率 | 呆滞库存 / 总库存 | < 2% |
| 库存准确率 | 盘点准确项 / 总项 | > 99% |
