# 合规检查审计

## 功能定位

确保集团业务运营符合外部监管法规和内部制度要求，识别风险，整改闭环。

## 合规框架

### 外部监管合规

| 监管领域 | 法规要求 | 主责部门 |
|----------|----------|----------|
| 海关 | 进出口申报准确、证件齐全 | 贸易执行部 |
| 外汇 | 外汇收支合规、跨境资金流动合规 | 财务部 |
| 税务 | 税务申报准确、优惠申请合规 | 财务部 |
| 安监 | 危化品经营资质、安全管理 | 仓储部 |
| 环保 | 油品仓储运输环保合规 | 仓储部 |

### 内部制度合规

- 授权审批：超权限业务需上级审批
- 合同管理：合同条款审核用印规范
- 资金管理：付款审批、收款核销规范
- 信息安全：客户数据保护、商业秘密

## 检查类型

### 1. 日常合规检查

**频率**：每日/每周

**检查内容**：
- 交易价格是否在授权范围内
- 付款是否符合审批流程
- 单据是否完整准确

### 2. 季度合规审查

**频率**：每季度

**审查内容**：
- 本季度业务合规情况
- 风险事件统计
- 整改措施落实情况
- 制度更新与培训情况

### 3. 年度合规审计

**频率**：每年

**审计范围**：
- 全面业务合规审查
- 内控体系有效性评估
- 外部审计配合

## 检查流程

```
1. 制定检查计划：确定检查范围、重点、时间
2. 发送检查通知：提前通知被检查部门
3. 收集资料：制度文件、操作记录、凭证
4. 现场检查：访谈、记录抽样、合规测试
5. 发现问题：不符合项编号、描述、证据
6. 整改要求：整改期限、整改责任人
7. 整改跟踪：跟踪整改进度
8. 关闭归档：确认整改完成，关闭检查项
```

## 检查清单示例

**贸易业务合规检查清单**：
- [ ] 合同审批流程完整
- [ ] 交易价格符合授权范围
- [ ] 客户信用额度未超限
- [ ] 交货期符合合同约定
- [ ] 品质争议处理规范

**财务合规检查清单**：
- [ ] 付款审批手续完整
- [ ] 发票与合同一致
- [ ] 外汇收支符合规定
- [ ] 税务申报准确

## 合规评级

| 评级 | 说明 | 后续措施 |
|------|------|----------|
| A | 无重大不符合项 | 持续监控 |
| B | 少量一般不符合项 | 限期整改 |
| C | 存在重大不符合项 | 暂停业务整改 |
| D | 严重违规 | 升级处理、法律行动 |

## 回复模板

- 检查通知：`📋 合规检查通知：{类型}检查将于{时间}开始，请准备{资料}`
- 检查完成：`✅ {部门}{类型}检查完成，评级：{评级}，发现问题{数量}项`
- 不符合项：`❌ 不符合项 #{编号}：{描述}，整改期限{期限}，责任人{人}`
- 整改完成：`✅ 不符合项 #{编号}已整改完成，审核通过`
- 审计报告：`📄 {年度}合规审计报告已完成，评级{评级}，详见附件`
