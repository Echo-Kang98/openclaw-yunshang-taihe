# 效能分析报告

## 功能定位

通过数据驱动分析，评估各部门/业务线运营效能，识别低效环节，提出改进建议。

## 分析维度

### 1. 贸易效能

| 指标 | 说明 | 分析方法 |
|------|------|----------|
| 人均贸易量 | 贸易量 / 业务人员 | 与行业对标 |
| 单笔贸易利润 | 净利润 / 订单数 | 与历史均值对标 |
| 客户开发效率 | 新客户数 / 销售费用 | 趋势分析 |

### 2. 运营效能

| 指标 | 说明 | 分析方法 |
|------|------|----------|
| 订单处理效率 | 订单处理时长 | 流程耗时分析 |
| 单据处理效率 | 单据处理时长 | 瓶颈分析 |
| 异常处理效率 | 异常处理时长 | 对比分析 |

### 3. 财务效能

| 指标 | 说明 | 分析方法 |
|------|------|----------|
| 资金使用效率 | 吨油占用资金 | 对标分析 |
| 回款效率 | 回款天数 | 客户维度分析 |
| 费用控制率 | 费用 / 贸易额 | 预算对比 |

## 分析方法

### 横向对比

- 各部门同一指标对比
- 各业务线同一指标对比
- 与行业平均/最优对比

### 趋势分析

- 同比分析：今年 vs 去年
- 环比分析：本月 vs 上月
- 移动平均：消除季节性波动

### 根因分析

- 5Why 分析法：层层追问根本原因
- 帕累托分析：识别关键少数因素
- 鱼骨图：系统性归因

## 报告结构
