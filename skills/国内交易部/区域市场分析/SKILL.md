# 区域市场分析

## 功能定位

分析国内各区域石油/大宗商品市场供需状况、价格差异、竞争格局，支持采购和销售决策。

## 分析维度

### 1. 区域市场供需分析

**华东市场**（长三角）
- 主要消费地：江苏、浙江、上海
- 主营产品：汽柴油、化工原料
- 供需特点：需求量大，供应充足，竞争激烈

**华南市场**（珠三角）
- 主要消费地：广东、福建
- 主营产品：汽柴油、LNG
- 供需特点：进口依赖度高，价格波动大

**华北市场**（环渤海）
- 主要消费地：北京、天津、河北、山东
- 主营产品：柴油、沥青
- 供需特点：炼化产能集中，供应充裕

**西南市场**
- 主要消费地：四川、云南、贵州
- 主营产品：柴油、天然气
- 供需特点：运输成本高，价格偏高

### 2. 区域价格分析

**价格监测**：
- 监测各省主要库区出库价
- 监测主流炼厂出厂价
- 监测主要批发市场价格

**价差分析**：
- 区域间价差：各省价格与均价差异
- 时间价差：区域价格历史走势
- 理论套利窗口：跨区域套利机会

### 3. 区域竞争格局

**主要玩家**：
- 国企：三桶油、地方炼厂
- 贸易商：各区域主要贸易商
- 终端：大型终端用户

**市场份额**：
- 各玩家市场占有率
- 各渠道销售占比（直销/分销）

### 4. 市场预测

**驱动因素**：
- 宏观经济：GDP增速、PMI
- 政策影响：环保政策、税收政策
- 季节性因素：旺季/淡季
- 替代能源：新能源影响

**预测方法**：
- 供需平衡表推算
- 价格历史走势外推
- 专家判断修正

## 报告输出

**日报**：各区域市场今日价格动态
**周报**：区域价差、套利机会分析
**月报**：区域供需总结、下月预测

## 回复模板

- 价格更新：`🗺 {区域}{品名}今日价格：{价格}元/吨，环比{变化}`
- 价差分析：`📊 区域价差：华东-华南{价差}元/吨，{存在/无}套利机会`
- 供需判断：`📈 {区域}市场供需：{偏紧/平衡/宽松}，主要原因：{原因}`
- 市场预测：`🔮 {区域}{品名}下月预测：均价{价格}元/吨，波动区间{区间}`
