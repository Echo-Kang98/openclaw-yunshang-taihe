# 国内市场情报收集

## 功能定位

系统收集和整理国内市场动态信息，为贸易决策提供及时、准确的情报支持。

## 情报类型

### 1. 价格情报

**监测渠道**：
- 国家发改委油价调整公告
- 中石化、中石油出厂价
- 卓创、隆众等资讯平台
- 各区域库区报价
- 期货盘面（INE原油/SC）

**价格数据项**：
| 数据项 | 更新频率 | 数据来源 |
|--------|----------|----------|
| 原油期货 | 实时 | INE |
| 山东地炼价格 | 每日 | 卓创资讯 |
| 主营单位价格 | 每日 | 官网公告 |
| 区域库区价格 | 每日 | 实地采集 |

### 2. 政策情报

**政策类型**：
- 价格政策：成品油定价机制
- 环保政策：限产、淘汰落后产能
- 贸易政策：进口配额、出口关税
- 安全政策：危化品运输/仓储规定

**政策影响评估**：
```
政策变化 → 影响供给/需求/成本 → 价格影响评估 → 业务应对策略
```

### 3. 供需情报

**供应端**：
- 主营炼厂开工率
- 地炼开工率
- 进口量/出口量
- 库存水平（商业库存、炼厂库存）

**需求端**：
- 终端消费情况
- 库存变化（业者库存）
- 出口订单

### 4. 竞争情报

**竞争对手监测**：
- 竞争对手价格策略
- 竞争对手市场动向
- 竞争对手新客户开发

**市场结构变化**：
- 新进入者
- 产能变化
- 渠道变化

### 5. 突发情报

**监测范围**：
- 重大事故（炼厂、仓储）
- 自然灾害（台风、洪水）
- 地缘政治事件
- 突发公共事件

**响应流程**：
```
情报识别 → 影响评估 → 预警发布 → 业务应对 → 后续跟踪
```

## 情报收集方法

| 方法 | 适用情报 | 频率 |
|------|----------|------|
| 公开数据采集 | 价格、政策 | 日 |
| 行业资讯订阅 | 行业动态 | 日 |
| 实地调研 | 市场一线 | 周 |
| 专家访谈 | 专业判断 | 月 |
| 内部数据 | 销售数据 | 日 |

## 情报报告

**日报**：每日市场动态、价格汇总
**周报**：本周市场分析、下周预测
**专题报告**：重大事件深度分析

## 回复模板

- 价格异动：`🔍 {品名}价格异动：今日{价格}，昨日{昨日}，涨幅{涨幅}%，原因：{原因}`
- 政策速递：`📢 {政策名称}：{核心内容}，影响：{影响分析}，建议：{应对}`
- 供需变化：`📊 {市场}供需：库存{库存}万吨，环比{变化}，{偏紧/平衡/宽松}`
- 竞争动态：`👀 竞争对手{名称}动态：{描述}，对我方影响：{影响}`
- 突发预警：`⚠️ 突发事件：{描述}，影响范围：{范围}，建议：{应对措施}`
