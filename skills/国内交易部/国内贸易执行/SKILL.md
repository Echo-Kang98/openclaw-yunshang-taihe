# 国内贸易执行

## 功能定位

执行国内石油/大宗商品贸易订单全流程，确保采销衔接、履约及时、结算准确。

## 执行流程

### 1. 订单确认

**触发**：与国内供应商/客户达成贸易意向

**确认事项**：
- 品名、规格、数量、单价
- 交割方式（自提/送到）
- 交割时间/周期
- 付款方式（预付/月结/现款）
- 发票类型（普票/专票）

### 2. 合同签订

**合同类型**：
- 年度框架协议：约定全年采购量、定价机制
- 现货合同：具体批次订单

**审核要点**：
- 价格与询价一致
- 数量与库存/产能匹配
- 付款方式可接受
- 履约条款明确

### 3. 采购执行

**采购模式**：
- 直采：向炼厂/供应商直接采购
- 贸易采购：向贸易商采购

**执行要点**：
- 确认供应商货源
- 确认交割时间
- 跟踪发货进度

### 4. 销售执行

**销售模式**：
- 预售：先接单后采购
- 库存销售：直接销售现有库存

**执行要点**：
- 客户确认订单
- 开单发货
- 送货跟踪

### 5. 物流协调

**物流方式**：
- 公路槽车：最灵活，适合小批量
- 铁路罐车：适合中距离
- 水路船运：适合大批量

**协调内容**：
- 运力安排
- 车辆/船舶调度
- 到货跟踪

### 6. 结算收付款

**结算流程**：
```
开票申请 → 发票开具 → 递送/邮寄 → 收款/付款 → 核销
```

**付款方式**：
- 先款后货：预付全款
- 货到付款：验收后付款
- 月结：月度汇总结算

## 异常处理

| 异常 | 处理方式 |
|------|----------|
| 价格波动 | 协商调整或取消订单 |
| 供应商断供 | 启动备选供应商 |
| 物流延误 | 与客户沟通延期，与物流协调加急 |
| 品质异议 | 协商降价、换货或退货 |
| 付款逾期 | 催款、停止发货、法律途径 |

## 执行指标

| 指标 | 目标 |
|------|------|
| 订单履约率 | > 98% |
| 交割准时率 | > 95% |
| 单据准确率 | > 99% |
| 客户投诉率 | < 1% |
