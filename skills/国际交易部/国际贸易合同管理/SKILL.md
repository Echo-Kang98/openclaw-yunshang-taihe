# 国际贸易合同管理

## 功能定位

管理国际贸易合同全生命周期，从起草到关闭归档，确保合同合规、风险可控、履约顺利。

## 合同起草

### 合同框架

**主合同内容**：
- 合同双方信息
- 货物品名、规格、数量、单价
- 总金额
- 贸易条款（Incoterms 2020）
- 装港、卸港
- 交货期/装运期
- 付款方式

### 贸易条款选择

| 条款 | 卖方责任 | 适用场景 |
|------|----------|----------|
| FOB | 装港船上交货 | 买方租船 |
| CFR | 运费付至卸港 | 卖方安排运输 |
| CIF | 运费+保险付至卸港 | 卖方安排运输+保险 |
| DAP | 目的地交货 | 卖方承担一切费用风险 |
| DDP | 完税后交货 | 卖方承担最大责任 |

### 品质数量条款

**品质标准**：
- 品牌/牌号
- 技术指标（如API度、含硫量）
- 检验标准（如ASTM）

**数量条款**：
- 装港数量（FOB）/ 卸港数量（CIF）
- 计量方式（流量计/岸罐/船舱）
- 溢短装条款（±5%）

## 合同审阅

### 法务审核

- 合同主体合法性
- 违约责任合理性
- 不可抗力条款
- 争议解决机制

### 业务审核

- 价格与市场水平
- 履约可行性
- 条款风险评估

### 审批流程

```
承办人 → 部门负责人 → 法务审核 → 财务审核 → 分管领导 → 签章
```

## 合同执行跟踪

### 关键节点

| 节点 | 时间要求 | 责任方 |
|------|----------|--------|
| 合同签订 | T+0 | 贸易部 |
| 信用证开立 | 装运期前30天 | 财务部 |
| 备货确认 | 装运期前7天 | 贸易部 |
| 装运通知 | 装船后24小时 | 供应商 |
| 单据提交 | 装船后5个工作日 | 供应商 |
| 付款赎单 | 收到单据7天内 | 财务部 |

### 执行监控

- 节点提醒：提前3天自动提醒
- 异常预警：节点延误告警
- 执行进度看板

## 合同变更与争议

### 合同变更

**可变更情形**：
- 双方协商一致
- 不可抗力导致延期
- 市场价格大幅波动（价格重议条款）

**变更流程**：
补充协议 → 审批 → 签章 → 归档

### 争议解决

**解决层次**：
1. 友好协商
2. 第三方调解
3. 仲裁（推荐）
4. 诉讼

**仲裁机构**：
- 国际油品：GAFTA、FOSFA（伦敦）
- 金属/其他：ICC（国际商会）

## 合同关闭

**关闭条件**：
- 货物交付完成
- 全额收付款项
- 单证归档完毕
- 无未解决争议

**归档要求**：
- 纸质合同原件
- 补充协议
- 往来函电
- 履约证明

## 回复模板

- 合同起草：`📄 合同草案已起草：{合同号}，{品名}{数量}{单位}，{价格条款}，请{审核}`
- 合同签订：`✅ 合同已签订：{合同号}，{卖方}→{买方}，金额{金额}，生效日期{日期}`
- 节点提醒：`⏰ {节点}节点提醒：截止{日期}，剩余{天数}天，请{事项}`
- 变更申请：`📋 合同{编号}变更申请：{变更内容}，原因：{原因}`
- 争议处理：`⚠️ {合同号}争议：{描述}，建议解决方式：{方式}`
- 合同关闭：`✅ 合同{编号}已关闭：履约完毕，{结算情况}`
