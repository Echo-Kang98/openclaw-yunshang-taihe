# 国际贸易执行

## 功能定位

执行国际贸易订单全流程，协调国际物流、单证、结算各环节，确保按时、按质、按量交付。

## 执行流程

### 1. 合同执行准备

**确认清单**：
- 合同正本收齐
- 信用证条款确认（如有）
- 供应商交货计划确认
- 船期/舱位确认

### 2. 信用证管理（如适用）

**开证申请**：
```
1. 提交开证申请：合同号、金额、信用证条款
2. 财务审核：开证额度、保证金
3. 银行开证：向通知行开立L/C
4. 传递受益人：通过银行传递L/C
```

**信用证条款审核**：
- 受益人名称、地址
- 货物描述与合同一致
- 装运期/有效期
- 单据要求
- 软条款识别

### 3. 国际物流执行

**装运准备**：
- 与供应商确认装运计划
- 提前7天确认船期
- 准备装运文件（装港委托）

**装运执行**：
- 装港锚地/泊位等待
- 货物装船
- 数量/品质计量
- 大副收据签发

**装运后处理**：
- 装运通知（Fax/Mail）
- 提单(B/L)获取
- 其他单据收集（发票、箱单、产地证等）

### 4. 单证管理

**正本单据清单**：
| 单据 | 份数 | 作用 |
|------|------|------|
| 提单 B/L | 3份正本 | 物权凭证 |
| 商业发票 INV | 3份 | 结算凭证 |
| 装箱单 P/L | 3份 | 包装明细 |
| 原产地证 CO | 1份 | 产地证明 |
| 商检证书 CIQ | 1份 | 品质/数量证明 |

**单据审核**：
- 单证一致
- 单货一致
- 提交银行/客户的单据齐全

### 5. 结算收付款

**T/T 结算流程**：
```
1. 供应商发货，提交装运通知+单据副本
2. 我方确认收款金额
3. 客户付款（T/T预付/到付）
4. 收到货款后，通知供应商寄送正本单据
5. 客户收到单据，提货
```

**L/C 结算流程**：
```
1. 供应商装运，提交全套单据给议付行
2. 议付行审核单据
3. 单据相符 → 议付行垫款
4. 我方赎单（付款或承兑）
5. 我方取得单据，提货
```

### 6. 到港清关

**清关准备**：
- 预报关：货物到港前申报
- 准备清关文件
- 安排卸港仓储

**清关执行**：
- 进口报关
- 关税缴纳
- 商检查验（如有）
- 海关放行

## 异常处理

| 异常 | 处理方式 |
|------|----------|
| 信用证单据不符 | 与供应商沟通修改，或与客户协商接受不符点 |
| 船期延误 | 申请信用证展期，通知客户 |
| 货物品质不符 | 协商降价、索赔或退货 |
| 海关查验 | 配合查验，提供补充资料 |
| 客户拒付 | 协商解决方案，法律途径 |

## 关键时效

| 时效 | 要求 |
|------|------|
| 装运通知 | 装船后24小时 |
| 单据提交 | 装船后5个工作日内 |
| 信用证有效期 | 装运后15-21天 |

## 回复模板

- 信用证开立：`📜 信用证已开立：L/C号{编号}，金额{金额}，开证行{银行}`
- 装运通知：`🚢 {品名}{数量}{单位}已装运：装港{装港}，离港{时间}，预计到港{时间}`
- 单据收到：`📄 已收到单据：提单{编号}、发票{编号}，正本{份数}份，审核{通过/问题}`
- 单据寄送：`📮 单据已寄出：DHL追踪号{编号}，预计{时间}到达`
- 客户赎单：`💰 客户已赎单：金额{金额}，已承兑/已付款`
- 清关完成：`🛃 清关已完成：货物已放行，存放{仓库}`
