# 外汇对冲策略管理

## 功能定位

管理国际业务中的外汇风险，通过对冲工具和策略设计，降低汇率波动对贸易利润的影响。

## 风险识别

### 主要敞口币种

| 币种 | 敞口类型 | 主要来源 |
|------|----------|----------|
| USD | 美元敞口 | 进口采购以USD计价 |
| EUR | 欧元敞口 | 欧洲设备进口 |
| JPY | 日元敞口 | 日本设备/技术进口 |

### 敞口类型

| 类型 | 说明 | 敞口方向 |
|------|------|----------|
| 交易敞口 | 已签订合同的应收应付 | 汇率不利变动导致损失 |
| 经济敞口 | 未来预期业务的敞口 | 未来收入/成本受影响 |
| 折算敞口 | 报表折算的汇率差 | 会计科目变动 |

### 风险评估指标

- 敞口金额：各币种净敞口
- 风险价值（VaR）：99%置信度下的最大日损失
- 敏感度：汇率1%变动对利润的影响

## 对冲工具

### 1. 远期外汇合约（Forward）

**特点**：
- 锁定未来交割汇率
- 无需初期成本
- 灵活性低（到期必须交割）

**适用场景**：
- 已签订合同，确定收款/付款日期
- 规避确定性敞口

### 2. 外汇期权（Option）

**类型**：
- 买入看涨期权（锁定最高采购成本）
- 买入看跌期权（锁定最低销售收入）

**特点**：
- 保留汇率有利变动的收益
- 需支付期权费

### 3. 外汇掉期（Swap）

**特点**：
- 同时进行即期卖出+远期买入
- 短期资金周转+汇率锁定

### 4. 货币互换（Cross-Currency Swap）

**特点**：
- 本金+利息交换
- 长期汇率风险管理

## 对冲策略设计

### 策略原则

1. **匹配原则**：对冲工具与敞口币种、期限匹配
2. **比例原则**：部分对冲（非100%），保留部分灵活性
3. **成本效益原则**：对冲成本 vs 风险敞口

### 比例确定

| 敞口确定性 | 对冲比例 | 工具选择 |
|------------|----------|----------|
| 确定合同 | 80-100% | 远期 |
| 已报价未签合同 | 50-80% | 远期+期权 |
| 预期业务 | 20-50% | 期权 |
| 投机性敞口 | 0% | 不对冲 |

### 对冲方案示例

**场景**：3个月后需支付1000万美元

**方案A（远期对冲）**：
```
卖出远期 USD/CNY 3个月
锁定汇率：7.25
到期支付 RMB 7250万 兑换 USD 1000万
```

**方案B（期权对冲）**：
```
买入 USD/CNY 看涨期权
执行价：7.30
期权费：0.05%（约3.6万RMB）
到期选择：若即期>7.30，行使期权；若<7.30，按即期购汇
```

## 执行管理

### 执行流程

```
1. 敞口识别 → 汇总各业务部门敞口
2. 对冲申请 → 部门提交对冲需求
3. 策略审批 → 财务/风控审批
4. 执行操作 → 银行询价，执行对冲
5. 持仓监控 → 每日盯市，盈亏评估
6. 到期处理 → 交割或展期
```

### 持仓监控

**每日评估**：
- 对冲合约市值
- 敞口变化
- 对冲有效性评估（套期保值比率）

**预警机制**：
- 汇率波动 > 1%：即时通知
- 对冲成本超预算：分析原因，调整策略

## 对冲效果评估

### 评估指标

| 指标 | 计算 | 目标 |
|------|------|------|
| 对冲有效率 | 对冲后波动 / 对冲前波动 | < 20% |
| 对冲成本率 | 对冲成本 / 敞口金额 | < 1% |
| 套期保值收益 | 对冲工具盈亏 vs 汇率变动 | 相互抵销 |

## 回复模板

- 敞口报告：`💱 外汇敞口：USD净敞口{金额}万，EUR净敞口{金额}万，风险评估{评估}`
- 对冲建议：`📋 对冲方案：{工具}，金额{金额}，期限{期限}，预计成本{成本}`
- 对冲执行：`✅ 对冲已执行：{工具}，汇率{汇率}，到期日{日期}`
- 持仓更新：`📊 对冲持仓市值：浮盈/浮亏{金额}，对冲有效率{效率}%`
- 对冲评估：`📈 对冲效果：本季对冲成本{成本}，规避汇率损失{损失}，净效果{效果}`
