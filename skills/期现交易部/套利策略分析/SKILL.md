# 套利策略分析

## 功能定位

识别石油/大宗商品市场套利机会，分析套利空间和风险，输出可执行的套利策略建议。

## 套利类型

### 1. 跨期套利（Calendar Spread）

**原理**：同品种不同合约月份价差偏离正常区间

**案例**：SC原油 2506 vs 2509
- 正常价差：-$2 ~ -$4（远月贴水）
- 当前价差：-$1.5（偏高）
- 策略：买入2506，卖出2509
- 目标价差：-$3

**分析要点**：
- 历史价差分布
- 近期供需预期（近月紧张→近月升水）
- 库存变化（去库→贴水加深）
- 跨月价差季节性

### 2. 跨品种套利

**原理**：相关品种价差偏离历史均值

**案例**：SC vs Brent
- 正常区间：$3 ~ $8（SC贴水Brent）
- 当前价差：$2（偏低）
- 策略：买入SC，卖出Brent
- 目标价差：$5

**常见跨品种对**：
- SC vs Brent
- SC vs WTI
- 原油 vs 燃料油
- 原油 vs 沥青

### 3. 跨市场套利

**原理**：同一品种在不同市场价格偏离

**案例**：
- 现货市场：华东现货 3800元/吨
- 期货市场：SC 3750元/吨
- 差差：现货升水50元
- 策略：买期货抛现货（买入SC期货，虚拟销售现货）

### 4. 期现套利

**原理**：期货与现货价差超出持有成本

**无风险套利条件**：
```
期货价格 > 现货价格 + 仓储成本 + 资金成本 + 其他成本
```

**操作**：
- 买入现货（或虚拟买入）
- 卖出期货合约
- 持仓至到期交割

## 分析框架

### 1. 数据收集

- 期货价格（收盘价、成交量、持仓量）
- 现货价格（主流市场价格）
- 价差历史数据（30日、90日、1年）
- 库存数据（期货交易所库存+商业库存）
- 仓单数量

### 2. 统计套利分析

**均值回归模型**：
```
Z-score = (当前价差 - 移动平均) / 标准差
|Z| > 2 → 套利机会
```

**季节性分析**：
- 各月份价差历史均值
- 当前价差在历史分布中位置

### 3. 基本面验证

**驱动因素检查**：
- 近月供需：紧张→近月升水
- 远月供需：宽松→远月贴水
- 库存：低库存→贴水收窄
- 运费：运价变化影响跨市场价差

## 风控要点

| 风控项 | 要求 |
|--------|------|
| 止损 | 价差逆向偏离X%止损 |
| 仓位 | 单边不超过总资金X% |
| 保证金 | 预留足够保证金防强平 |
| 流动性 | 优先主力合约 |
