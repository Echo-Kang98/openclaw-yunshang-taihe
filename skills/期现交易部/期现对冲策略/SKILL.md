# 期现对冲策略

## 功能定位

为企业的现货采购/销售敞口设计期货对冲方案，通过期货市场对冲现货价格风险，实现稳健经营。

## 对冲场景

### 1. 采购对冲（买保）

**场景**：企业需在现货市场采购，担心价格上涨

**策略**：买入期货，锁定采购成本

**示例**：
- 企业3个月后需采购10000吨原油
- 当前现货价：$80/桶
- 买入3个月期货（SC）价格：$81/桶
- 3个月后：现货涨至$90，期货涨至$91
- 期货盈利：$10/桶 × 10000 = $100,000
- 实际采购成本：$90 - $10 = $80（锁定了采购价）

### 2. 销售对冲（卖保）

**场景**：企业有库存或远期销售合同，担心价格下跌

**策略**：卖出期货，锁定销售利润

**示例**：
- 企业有10000吨库存，当前价$80/桶
- 卖出3个月期货（SC）价格：$81/桶
- 3个月后：现货跌至$70，期货跌至$71
- 期货盈利：$10/桶 × 10000 = $100,000
- 实际销售价格：$70 + $10 = $80（锁定了销售价）

## 对冲方案设计

### 1. 敞口识别

**敞口类型**：
| 类型 | 说明 | 对冲方向 |
|------|------|----------|
| 已定价敞口 | 已购入未定价 | 卖出对冲 |
| 未定价敞口 | 计划采购/销售 | 双向灵活 |
| 库存敞口 | 常备库存 | 卖出对冲 |

**敞口量化**：
```
对冲敞口量 = 现货量 × 套保比率
```

### 2. 套保比率确定

**完全套保**：套保比率=100%
- 优点：完全锁定价格
- 缺点：失去价格有利变动的收益

**部分套保**：套保比率<100%
- 优点：保留部分价格弹性
- 缺点：仍有一定风险敞口

**比率确定因素**：
- 风险偏好（保守→高套保比率）
- 资金成本（保证金占用）
- 市场判断（短期方向）

### 3. 合约选择

**主力合约选择**：
- 优先选择主力合约（成交量最大）
- 考虑到期月份与对冲期间匹配
- 避免流动性差的合约

**备用方案**：
- 主力合约流动性不足时，选择次主力
- 考虑使用价差合约（spread）

### 4. 动态调整

**调整触发条件**：
- 敞口数量变化（采购/销售完成）
- 套保比率调整（增加/减少对冲）
- 移仓需求（合约到期前）

**移仓操作**：
- 平仓近月合约
- 开仓远月合约
- 注意移仓成本（价差损失）

## 对冲效果评估

### 评估指标

| 指标 | 计算 | 说明 |
|------|------|------|
| 对冲效率 | (期货盈亏 / 现货损失) × 100% | 理想>90% |
| 基差风险 | 基差变动 × 对冲量 | 未对冲部分 |
| 套保损益 | 期货盈亏 + 现货盈亏 | 综合效果 |

### 基差分析

**基差** = 现货价格 - 期货价格

**基差风险**：
- 期货与现货价格变动不完全同步
- 套保后仍保留基差风险

**基差走强/走弱影响**：
- 买方套保：基差走强→对冲效果更好
- 卖方套保：基差走弱→对冲效果更好

## 对冲操作流程

```
1. 敞口识别 → 量化对冲需求
2. 方案设计 → 确定套保比率、合约
3. 审批执行 → 交易员执行下单
4. 持仓监控 → 每日盯市、评估效果
5. 到期处理 → 交割/平仓/移仓
6. 效果评估 → 对冲效果分析总结
```
