# 期货账户风险管理

## 功能定位

实时监控期货账户风险，确保持仓安全，及时预警和处理风险事件。

## 核心风险指标

### 1. 保证金监控

**保证金水平**：
- 持仓保证金 = 合约数 × 合约价值 × 保证金率
- 可用资金 = 账户余额 - 持仓保证金
- 风险度 = 持仓保证金 / 账户权益 × 100%

**风险等级**：
| 风险度 | 等级 | 措施 |
|--------|------|------|
| < 50% | 🟢 正常 | 正常交易 |
| 50-80% | 🟡 关注 | 限制新建仓 |
| 80-100% | 🟠 预警 | 追保通知 |
| > 100% | 🔴 危险 | 强平执行 |

### 2. 持仓限额

**期货公司限制**：
- 单账户单合约持仓上限
- 单账户总持仓上限

**交易所限制**：
- 套保持仓 vs 投机持仓限额不同
- 超仓需平仓或申请豁免

### 3. 清算是指

**到期管理**：
- 主力合约换月：提前5个交易日提示
- 临近交割：自然人户需平仓

**浮动盈亏**：
- 盯市浮盈/亏 → 计入可用资金
- 浮亏过大 → 追保通知

## 监控流程

### 每日监控

```
开盘前检查：
1. 账户余额是否充足
2. 持仓是否超限
3. 风险度是否正常

盘中监控：
1. 价格波动 → 风险度变化
2. 异常波动 → 预警通知
3. 新建仓 → 风险度检查

收盘后：
1. 结算 → 计算当日盈亏
2. 持仓市值更新
3. 生成日报
```

### 预警机制

**价格波动预警**：
- 单日涨跌幅 > 5%：即时通知
- 连续3日涨跌停：交易限制

**保证金预警**：
- 风险度 > 80%：追保通知
- 风险度 > 100%：强制平仓

**持仓预警**：
- 临近交割月：平仓提示
- 超仓：立即平仓

## 清算管理

### 强行平仓

**触发条件**：
- 风险度持续 > 100%
- 会员未按时追保
- 交易所规定的其他情形

**平仓原则**：
1. 先平投机仓，后平套保仓
2. 先平亏损最大的持仓
3. 尽可能选择流动性好的合约

### 移仓管理

**操作流程**：
```
1. 发出移仓指令
2. 平仓近月合约
3. 开仓远月合约
4. 确认移仓后持仓
5. 更新风险数据
```

## 日报生成

### 期货日报内容

```
日期：YYYY-MM-DD

一、账户概况
- 账户余额：XXX万元
- 持仓保证金：XXX万元
- 可用资金：XXX万元
- 风险度：XX%

二、持仓明细
| 合约 | 方向 | 手数 | 开仓均价 | 昨结算 | 今收盘 | 浮动盈亏 |
|------|------|------|----------|--------|--------|----------|

三、交易记录
| 时间 | 合约 | 方向 | 手数 | 价格 | 原因 |
|------|------|------|------|------|------|

四、风险事件
- 无

五、下日关注
- 关注SC2506合约移仓窗口
```

## 回复模板

- 账户概览：`⚠️ 账户风险度{风险度}%，{状态}，可用资金{可用}万`
- 追保通知：`💰 追保通知：风险度{风险度}%，需追加保证金{金额}万，期限{时间}`
- 强平执行：`🔴 强平执行：{合约}平仓{手数}手，成交均价{价格}，亏损{金额}`
- 持仓预警：`⚠️ 持仓预警：{合约}{方向}已达持仓上限{手数}手，需平仓或申请豁免`
- 日报生成：`📊 期货日报已生成：{日期}，账户余额{余额}万，风险度{风险度}%`
