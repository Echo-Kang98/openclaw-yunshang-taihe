# 套利窗口监测

## 功能定位

监测外汇市场套利机会，包括利率平价、离在岸价差、跨币种Carry Trade等，为汇率经营部提供交易机会支持。

## 套利类型

### 1. 利率平价套利（Interest Rate Parity）

**原理**：利用不同货币的利率差异，通过远期外汇市场锁定收益

**计算公式**：
```
远期汇率 = 即期汇率 × (1 + r本币) / (1 + r外币)

若实际远期汇率 ≠ 理论远期汇率 → 套利机会
```

**示例**：
- USD利率 5%，CNY利率 2%
- 即期 USD/CNY = 7.20
- 理论远期（1年）= 7.20 × 1.05/1.02 = 7.41
- 市场远期 = 7.38
- 套利空间 = 7.41 - 7.38 = 0.03（3分）

**监测指标**：
- 各币种利率（SHIBOR/LIBOR）
- 远期汇率报价（1W/1M/3M/6M/1Y）
- 理论价 vs 市场价偏离度

### 2. 离在岸套利

**原理**：CNH（离岸）与CNY（在岸）价差超过正常区间

**正常价差区间**：
- 日间：±50bp
- 极端：±200bp（特殊时期）

**套利操作**：
- CNY > CNH：买入CNH（离岸），卖出CNY（在岸）
- CNY < CNH：买入CNY（在岸），卖出CNH（离岸）

**触发条件**：
- 价差 > 100bp：套利窗口开启
- 价差 < 50bp：套利空间不足

### 3. 跨币种Carry Trade

**原理**：借入低息货币，换成高息货币存款，获取利差收益

**常用组合**：
- 借入JPY，兑换CNY/USD存款
- 借入USD，兑换新兴市场货币存款

**风险**：汇率反向变动侵蚀利差收益

### 4. 期权波动率套利

**原理**：期权隐含波动率 vs 实际波动率偏离

**套利方式**：
- 隐含波动率 > 实际波动率：卖出期权（收取期权费）
- 隐含波动率 < 实际波动率：买入期权（支付期权费）

## 监测指标

### 数据采集

| 指标 | 来源 | 更新频率 |
|------|------|----------|
| 即期汇率 | 中国外汇交易中心 | 实时 |
| 远期汇率 | 掉期点 | 实时 |
| SHIBOR | 上海银行间市场 | 每日11:30 |
| LIBOR/SOFR | 国际市场 | 每日 |
| CNH Hibor | 香港财资市场 | 实时 |
| 在岸人民币汇率 | 人行中间价 | 每日9:15 |

### 预警阈值

| 套利类型 | 预警阈值 | 处理 |
|----------|----------|------|
| 利率平价 | 偏离 > 30bp | 机会确认 |
| 离在岸 | 价差 > 100bp | 机会确认 |
| Carry Trade | 利差 > 150bp | 机会确认 |

## 策略输出

### 监测日报

```
# 套利窗口监测日报
日期：YYYY-MM-DD

一、利率平价监测
- USD/CNY即期：7.20
- 理论远期(1Y)：7.41
- 市场远期(1Y)：7.38
- 偏离度：-0.03（套利空间不足）

二、离在岸监测
- CNY在岸：7.2005
- CNH离岸：7.1980
- 价差：-25bp（正常区间）

三、Carry Trade监测
- USD融资成本：5.2%
- CNY存款利率：2.0%
- 综合利差：3.2%（年化）

四、机会评估
- 当前无明显套利窗口
- 建议关注：CNH Hibor明日利率变化
```

## 风控要点

- 交易成本：点差、手续费需纳入计算
- 流动性风险：市场极端时难以平仓
- 政策风险：外汇管制可能限制套利
- 汇率风险：套利持仓期间汇率不利变动
