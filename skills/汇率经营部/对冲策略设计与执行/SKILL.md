# 对冲策略设计与执行

## 功能定位

为企业的外汇敞口设计系统性的对冲策略，从识别敞口到执行对冲，全流程管理汇率风险。

## 敞口识别

### 敞口分类

| 类型 | 特征 | 对冲紧迫性 |
|------|------|------------|
| 已确认敞口 | 已签合同，收支确定 | 高 |
| 报价敞口 | 已报价未签合同 | 中 |
| 预期敞口 | 计划业务，预计发生 | 低 |

### 敞口量化

**美元敞口示例**：
```
收入敞口：+500万美元（3个月后应收）
支出敞口：-300万美元（3个月后应付）
净敞口：+200万美元（美元净多头）
风险：美元贬值 → 换回人民币减少
```

### 敞口报告

```
日期：YYYY-MM-DD

币种：USD/CNY

| 业务类型 | 敞口金额 | 期限 | 敞口方向 |
|----------|----------|------|----------|
| 已签合同-收入 | +500万 | 3个月 | 美元升值受益 |
| 已签合同-支出 | -300万 | 3个月 | 美元贬值受益 |
| 净敞口 | +200万 | 3个月 | 美元净多头 |
```

## 策略设计

### 对冲工具选择

| 工具 | 适用场景 | 优点 | 缺点 |
|------|----------|------|------|
| 远期外汇 | 确定敞口 | 锁定汇率，无成本 | 失去有利变动收益 |
| 外汇期权 | 需保留灵活性 | 保留上涨收益 | 需支付期权费 |
| 外汇掉期 | 短期周转+锁定 | 灵活 | 复杂 |
| 货币互换 | 长期资产/负债 | 长期锁定 | 规模要求大 |

### 对冲比例确定

| 敞口确定性 | 推荐比例 | 工具 |
|------------|----------|------|
| 100%确定（已签合同） | 80-100% | 远期 |
| 80%确定（订单确认） | 60-80% | 远期+期权 |
| 50%确定（报价中） | 30-50% | 期权 |
| 预期业务 | 0-20% | 观察 |

### 策略方案模板

```
# 对冲策略方案

一、敞口概述
- 币种：USD/CNY
- 净敞口：200万美元
- 期限：3个月
- 敞口类型：美元净多头

二、对冲目标
- 目标锁定汇率：7.20
- 可接受波动范围：±0.05
- 对冲成本预算：< 5万人民币

三、策略选择
- 主策略：远期外汇（锁定汇率）
- 辅助策略：买入看跌期权（保留上涨空间）
- 综合对冲比率：90%

四、执行计划
- 第一批：立即执行50%（锁定100万）
- 第二批：一周后执行30%
- 第三批：视市场情况执行10%

五、风险评估
- 汇率变动风险：有限（已锁定）
- 流动性风险：低（市场深度好）
- 信用风险：低（银行履约）
```

## 执行管理

### 执行流程

```
1. 方案审批 → 风控/财务审批
2. 银行询价 → 向2-3家银行询价
3. 执行交易 → 选择最优报价执行
4. 持仓确认 → 确认成交单据
5. 台账登记 → 登记对冲台账
6. 持仓监控 → 每日盯市
```

### 询价要点

- 同时向2-3家银行询价
- 明确金额、期限、货币对
- 记录询价时间、市场状况
- 选择最优报价执行

### 成交确认

**确认要素**：
- 交易对手
- 合约金额
- 汇率水平
- 交割日期
- 合约类型（远期/期权等）

## 效果评估

### 评估指标

| 指标 | 计算 | 评价标准 |
|------|------|----------|
| 对冲效率 | (对冲后波动/对冲前波动) × 100% | < 20% 为好 |
| 对冲成本 | 期权费等直接成本 | < 0.5% 为好 |
| 绝对收益率 | 对冲后综合收益率 | vs 基准 |

### 评估报告

```
# 对冲效果评估报告

期间：Q1 2026

一、对冲概况
- 对冲笔数：12笔
- 对冲金额：2000万美元
- 对冲期限：1-6个月
- 使用工具：远期80%，期权20%

二、效果评估
- 敞口汇率波动：±3%
- 对冲后汇率波动：±0.5%
- 对冲效率：83%
- 对冲成本：15万人民币

三、结论
对冲策略有效控制了汇率风险，对冲效率达到83%。
```
