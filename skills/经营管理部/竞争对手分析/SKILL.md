# 竞争对手分析

## 功能定位

建立竞争对手监测和分析体系，系统跟踪竞争对手动态，为竞争策略制定提供决策支持。

## 竞争对手识别

### 主要竞争对手分类

| 类型 | 定义 | 示例 |
|------|------|------|
| 直接竞争 | 同类产品、同类市场 | 中石化、振华石油 |
| 间接竞争 | 替代产品 | 新能源企业 |
| 潜在竞争 | 可能进入者 | 行业外巨头 |

### 竞争对手档案

**基本信息**：
- 公司名称、性质（国企/民营/外企）
- 注册资本、成立时间
- 总部位置、分支机构
- 股权结构、实际控制人

**业务概况**：
- 主营产品
- 销售规模
- 主要市场区域
- 商业模式

## 信息收集

### 公开信息来源

| 信息类型 | 来源 | 更新频率 |
|----------|------|----------|
| 财务数据 | 年报、季报 | 年度/季度 |
| 新闻动态 | 企业官网、媒体 | 实时 |
| 监管信息 | 工商、海关、交易所 | 定期 |
| 行业报告 | 咨询公司 | 定期 |

### 实地信息收集

- 客户反馈：了解竞争对手服务水平
- 行业展会：观察产品、展示、策略
- 同行交流：行业会议中的信息收集

## 分析维度

### 1. 战略分析

- 战略定位：竞争对手追求什么？
- 战略举措：最近有何重大战略动作？
- 战略意图：未来可能进入的市场？

### 2. 产品分析

| 维度 | 分析内容 |
|------|----------|
| 产品组合 | 产品线宽度、深度 |
| 产品质量 | 品质标准、技术指标 |
| 产品价格 | 价格定位、价格策略 |
| 产品创新 | 新产品研发、上市节奏 |

### 3. 市场行为分析

- 营销策略：促销、广告、渠道
- 价格行为：降价、涨价、促销
- 市场拓展：新市场进入、新渠道开发
- 客户政策：信用政策、服务政策

### 4. 财务分析

| 指标 | 分析用途 |
|------|----------|
| 营收规模 | 市场规模估算 |
| 毛利率 | 成本竞争力 |
| 净利率 | 盈利能力 |
| 资产负债率 | 财务杠杆 |

## 竞争对比表

```
| 维度       | 我方   | 竞争对手A | 竞争对手B |
|------------|--------|------------|------------|
| 市场份额   | 15%    | 20%        | 12%        |
| 价格定位   | 中高端 | 高端       | 中低端     |
| 渠道覆盖   | 全国   | 华东强     | 华南强     |
| 服务响应   | 24h    | 48h        | 72h        |
| 毛利率     | 8%     | 10%        | 5%         |
```

## 竞争预警

### 预警信号

| 信号类型 | 预警内容 |
|----------|----------|
| 新产品 | 竞争对手推出革命性产品 |
| 价格战 | 大幅降价抢市场份额 |
| 渠道战 | 大举进入我方优势区域 |
| 人才战 | 高薪挖角核心团队 |

### 应对预案

```
预警触发 → 评估影响 → 制定应对 → 高管审批 → 执行
```

## 回复模板

- 竞品档案：`🔎 竞争对手{名称}档案已更新：{最新动态}`
- 对比报告：`📊 竞品对比：{维度}我方{我方}，竞争对手A{对手A}，竞争对手B{对手B}`
- 动态追踪：`👀 {竞争对手}近30天动态：{动态1}、{动态2}、{动态3}`
- 竞争预警：`⚠️ 竞争预警：{竞争对手}{预警内容}，影响：{影响}，建议：{应对}`
