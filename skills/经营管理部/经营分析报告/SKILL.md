# 经营分析报告

## 功能定位

定期编制经营分析报告，全面反映集团经营状况，识别问题，提出改进建议，为管理层决策提供支持。

## 报告类型

### 月度经营分析

**发布时间**：每月结束后5个工作日内

**内容结构**：
```
一、主要指标完成情况
二、各业务线经营分析
三、重点问题分析
四、下月工作重点
```

### 季度经营分析

**发布时间**：每季度结束后10个工作日内

**内容结构**：
```
一、主要指标完成情况
二、各业务线经营分析
三、重大事项回顾
四、风险事件分析
五、下季工作重点
```

### 年度经营分析

**发布时间**：年度结束后20个工作日内

**内容结构**：
```
一、年度经营成果
二、各业务线经营分析
三、财务状况分析
四、战略执行评估
五、重大经验教训
六、明年工作规划
```

### 预算完成分析

```
预算完成率 = 实际完成 / 预算目标 × 100%

偏差分析 = 实际 - 预算
偏差率 = (实际 - 预算) / 预算 × 100%
```

## 业务分析框架

### 销量分析

**同比增长**：
```
同比增长率 = (本期销量 - 上年同期销量) / 上年同期销量 × 100%
```

**环比增长**：
```
环比增长率 = (本期销量 - 上期销量) / 上期销量 × 100%
```

### 毛利分析

```
毛利 = 销量 × (售价 - 进价)
毛利率 = 毛利 / 销售收入 × 100%

毛利率变化因素：
- 产品结构变化
- 市场定价变化
- 采购成本变化
```

## 问题诊断

### 问题识别

| 问题类型 | 识别标准 | 示例 |
|----------|----------|------|
| 预算偏差 | 偏差率>10% | 收入未达预算 |
| 趋势异常 | 连续3月下降 | 毛利持续下滑 |
| 竞品压力 | 市场份额下降 | 流失客户 |

### 根因分析

**方法**：5Why分析法

```
问题：月度贸易量未达预算
Why1：销售订单减少 → 为什么？
Why2：客户采购计划缩减 → 为什么？
Why3：竞争对手价格更低 → 为什么？
Why4：竞争对手获取了新货源 → 为什么？
Why5：缺乏长期供应商关系 → 根本原因
```
