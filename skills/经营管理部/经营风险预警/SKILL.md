# 经营风险预警

## 功能定位

建立经营风险预警体系，识别和预警各类经营风险，推动风险处置，防范重大损失。

## 风险类型

### 1. 市场风险

| 风险类型 | 监控指标 | 预警阈值 |
|----------|----------|----------|
| 价格风险 | 商品价格波动 | 日涨跌>5% |
| 汇率风险 | 汇率波动 | 日波动>1% |
| 流动性风险 | 库存周转天数 | >目标30天 |

### 2. 信用风险

| 风险类型 | 监控指标 | 预警阈值 |
|----------|----------|----------|
| 客户信用 | 应收账款逾期率 | >5% |
| 客户集中度 | 单客户销售占比 | >30% |
| 客户财务 | 客户重大负面 | 发生即为预警 |

### 3. 操作风险

| 风险类型 | 监控指标 | 预警阈值 |
|----------|----------|----------|
| 交易错误 | 差错笔数 | >2笔/月 |
| 合规风险 | 合规检查不合格项 | >3项/季度 |
| 系统风险 | 系统故障时间 | >2小时/月 |

### 4. 战略风险

| 风险类型 | 监控指标 | 预警阈值 |
|----------|----------|----------|
| 市场地位 | 市场份额变化 | 下降>5% |
| 竞争压力 | 竞争对手重大动作 | 发生即为预警 |
| 政策变化 | 行业政策重大调整 | 发生即为预警 |

## 预警体系

### 预警等级

| 等级 | 标识 | 触发条件 | 响应要求 |
|------|------|----------|----------|
| 一级 | 🔴 红色 | 严重偏离或事件发生 | 即时响应，高管介入 |
| 二级 | 🟠 橙色 | 显著偏离 | 24小时内响应 |
| 三级 | 🟡 黄色 | 轻度偏离 | 3日内响应 |
| 四级 | 🟢 蓝色 | 接近阈值 | 关注，无需行动 |

### 预警流程

```
1. 数据采集 → 自动抓取或人工录入
2. 阈值比对 → 系统自动比对阈值
3. 预警生成 → 符合条件则生成预警
4. 预警推送 → 通过系统/邮件/短信通知
5. 响应处置 → 责任人响应并处理
6. 预警关闭 → 处理完成后关闭
```

## 重点预警场景

### 场景1：客户信用风险

**触发条件**：
- 客户逾期超30天
- 客户经营出现重大异常
- 客户涉及诉讼

**响应流程**：
```
预警 → 暂停新订单 → 评估应收账款可回收性 → 制定催收方案 → 执行催收/诉讼
```

### 场景2：市场价格异常

**触发条件**：
- 单日价格涨跌超5%
- 连续3日同向变动

**响应流程**：
```
预警 → 评估对现有库存/订单影响 → 决定是否对冲 → 调整销售/采购策略
```

### 场景3：供应商断供风险

**触发条件**：
- 供应商重大事故（火灾、政治动荡）
- 供应商资金链紧张
- 供应商违约历史

**响应流程**：
```
预警 → 评估库存可用天数 → 启动备选供应商 → 必要时涨价抢货
```

## 预警管理

### 台账管理

```
| 预警编号 | 预警时间 | 风险类型 | 预警内容 | 等级 | 责任人 | 处置状态 | 关闭时间 |
```

### 复盘分析

**触发条件**：
- 预警关闭后1周内
- 重大预警（红色/橙色）

**复盘内容**：
- 预警处置过程回顾
- 预警有效性评估
- 阈值调整建议
- 预防措施

## 回复模板

- 预警触发：`🚨 {类型}预警：{内容}，等级{等级}，时间{时间}`
- 预警升级：`🔴 预警升级：{编号}从{原等级}升级为{新等级}，原因：{原因}`
- 预警响应：`✅ 已收到预警，责任人{姓名}，预计响应时间{时间}`
- 预警处置：`📋 预警{编号}处置中：{当前措施}，预计{时间}关闭`
- 预警关闭：`✅ 预警{编号}已关闭：处置结果{结果}，复盘结论{结论}`
- 预警统计：`📊 {期间}预警统计：共{总数}条，红色{红}条，橙色{橙}条，黄色{黄}条`
