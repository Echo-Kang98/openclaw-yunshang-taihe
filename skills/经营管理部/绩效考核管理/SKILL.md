# 绩效考核管理

## 功能定位

建立和维护绩效考核体系，包括指标设定、过程跟踪、结果评估，确保绩效管理公正、有效，支撑战略目标达成。

## 考核体系

### 考核周期

| 周期 | 适用范围 | 主要内容 |
|------|----------|----------|
| 年度考核 | 全体员工 | 综合绩效评定 |
| 半年度考核 | 中层管理人员 | 重点指标评估 |
| 季度考核 | 业务部门 | 业务指标跟踪 |
| 月度考核 | 业务部门 | 月度KPI |

### 考核维度

**分层考核**：

| 层级 | 考核维度 | 权重 |
|------|----------|------|
| 高管 | 战略指标 | 战略目标达成 60%，管理指标 40% |
| 中层 | 部门指标+管理 | 部门业绩 70%，管理能力 30% |
| 基层 | 岗位指标+能力 | 岗位KPI 80%，能力发展 20% |

## KPI体系

### KPI设计原则

| 原则 | 说明 |
|------|------|
| SMART | 具体、可衡量、可达成、相关、有时限 |
| 上下对齐 | 个人KPI支撑部门KPI |
| 平衡导向 | 短期+长期、财务+非财务 |

### KPI类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 结果型 | 产出结果导向 | 销售额、利润 |
| 过程型 | 过程行为导向 | 客户满意度 |
| 能力型 | 能力发展导向 | 技能提升 |

## 考核流程

### 年度考核流程

```
1. 目标设定（1月）→ 部门/岗位KPI确认
2. 过程跟踪（月度/季度）→ 绩效辅导与反馈
3. 年中检视（7月）→ KPI完成评估，调整
4. 年终考核（12月）→ 综合评定
5. 结果应用（次年1月）→ 薪酬调整、晋升、培训
```

### 考核评定

**评定等级**：

| 等级 | 定义 | 比例控制 |
|------|------|----------|
| A | 卓越，超额完成 | 10-15% |
| B+ | 优秀，高质量完成 | 20-25% |
| B | 良好，完成预期 | 40-50% |
| C | 合格，基本达标 | 10-15% |
| D | 不达标 | 5-10% |

### 强制分布

**比例限制**：
- A级：不超过15%
- B+级：不超过25%
- C级以下：不低于15%

## 绩效面谈

### 面谈内容

**结构化面谈**：
1. 开场：说明目的、流程
2. 员工自评：成绩、不足、改进
3. 主管评价：肯定成绩、指出不足
4. 共同制定：下期目标、改进计划
5. 结束：确认共识、鼓励

### 绩效反馈

```
面谈要点：
1. 事实依据：具体事例
2. 双向沟通：倾听员工想法
3. 建设性反馈：指出改进方向
4. 发展建议：培训、轮岗等
```

## 结果应用

### 薪酬激励

| 应用 | 说明 |
|------|------|
| 年终奖金 | 绩效等级决定系数 |
| 薪酬调整 | A/B+级优先调薪 |
| 特别奖励 | 突出贡献额外奖励 |

### 人才发展

| 应用 | 说明 |
|------|------|
| 晋升 | A/B+级优先考虑 |
| 培训 | 按需制定培训计划 |
| 轮岗 | 跨部门历练机会 |

## 回复模板

- KPI确认：`🏆 KPI确认：{期间}{对象}KPI共{数量}项，请{时间}前确认`
- 绩效进度：`📊 KPI进度：已完成{完成数}/{总数}项，达成率{达成率}%`
- 面谈安排：`📅 绩效面谈安排：{时间}，对象{对象}，请准备{内容}`
- 考核结果：`🏆 考核结果：{对象}{等级}，{评价}，奖金系数{系数}`
- 绩效改进：`💡 绩效改进计划：{目标}，措施{措施}，期限{期限}`
