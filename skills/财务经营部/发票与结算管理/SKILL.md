# 发票与结算管理

## 功能定位

管理发票开具、收票核对、结算付款全流程，确保财务合规、款项及时收付。

## 发票管理

### 发票开具

**开具条件**：
- 货物已交付或服务已提供
- 合同/订单已确认
- 价格与合同一致

**开具流程**：
```
1. 收到开票申请：客户名称、税号、金额
2. 核对信息：品名、税率、金额
3. 开具发票：增值税发票/普通发票
4. 发票审核：复核无误
5. 发票交付：寄送客户/自取
6. 归档：发票复印件存档
```

**税率说明**：
| 业务类型 | 税率 |
|----------|------|
| 原油 | 13% |
| 成品油 | 13% |
| 天然气 | 9% |
| 仓储服务 | 6% |
| 运输服务 | 9% |

### 收票管理

**收票审核**：
- 发票真实性
- 票面信息准确性
- 税率适用正确性
- 与合同/入库单匹配

**问题处理**：
| 问题类型 | 处理方式 |
|----------|----------|
| 发票信息错误 | 退回重开 |
| 税率错误 | 协商重开或折让 |
| 发票过期 | 确认是否可抵扣 |

## 结算管理

### 应收结算（向客户收款）

**结算流程**：
```
1. 确认应收：货物交付，确认应收金额
2. 开具发票：开具增值税发票
3. 发票交付：寄送客户
4. 到期提醒：付款到期前提醒
5. 收款确认：确认款项到账
6. 核销处理：收款核销应收
```

**账期管理**：
| 客户等级 | 账期 | 信用额度 |
|----------|------|----------|
| VIP | 30天 | >500万 |
| A级 | 15天 | 100-500万 |
| B级 | 7天 | 20-100万 |
| C级 | 0天 | <20万 |

### 应付结算（向供应商付款）

**结算流程**：
```
1. 收到发票：供应商提供发票
2. 收票审核：核对品名、数量、价格
3. 确认付款：与合同/入库单核对
4. 提交付款：提交付款申请
5. 审批执行：财务/领导审批
6. 付款执行：网银/转账
7. 核销处理：付款核销应付
```

**付款审批层级**：
| 金额范围 | 审批人 |
|----------|--------|
| <10万 | 部门负责人 |
| 10-50万 | 财务负责人 |
| 50-200万 | 分管副总 |
| >200万 | 总经理 |

## 对账管理

### 客户对账

**月度对账**：
- 发送客户对账单
- 核对应收余额
- 确认往来明细

**差异处理**：
- 查出差异原因
- 调整账务或联系客户

### 供应商对账

**核对内容**：
- 发票金额
- 入库数量
- 付款记录

## 回复模板

- 发票开具：`🧾 发票已开具：票号{编号}，客户{客户}，金额{金额}，税额{税额}，已寄送`
- 收票确认：`📥 发票已收：票号{编号}，供应商{供应商}，金额{金额}，核对{通过/问题}`
- 应收确认：`💰 应收确认：客户{客户}，金额{金额}，到期日{日期}`
- 付款执行：`✅ 付款完成：收款方{供应商}，金额{金额}，用途{用途}`
- 对账完成：`📋 对账完成：{客户/供应商}，期末余额{余额}，差异{差异}，{已处理/待处理}`
