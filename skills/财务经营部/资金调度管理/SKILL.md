# 资金调度管理

## 功能定位

管理集团资金流转，优化资金使用效率，防控资金风险，确保资金安全。

## 资金监控

### 资金池管理

**账户结构**：
- 总部一级账户
- 子公司二级账户
- 专用账户（期货保证金、托管账户）

**资金归集**：
- 子公司每日回流
- 归集比例设定
- 归集时间窗口

### 资金监控日报

**日报内容**：
```
日期：YYYY-MM-DD

一、账户余额
| 账户 | 币种 | 余额 | 折合人民币 |
|------|------|------|------------|
| 汇总 | CNY | xxx | xxx |

二、资金变动
- 昨日余额：xxx
- 本日收入：xxx
- 本日支出：xxx
- 本日余额：xxx

三、到期提醒
- 银行承兑汇票到期：xxx万元
- 信用证到期：xxx万元

四、异常预警
- 大额支出审批缺失
- 账户透支预警
```

## 闲置资金管理

### 闲置识别

```
闲置资金 = 账户余额 - 安全库存 - 已审批支出

安全库存 = 日常运营所需资金（通常为月支出的20-30%）
```

### 闲置资金处理

| 处理方式 | 说明 | 收益率 |
|----------|------|--------|
| 通知存款 | 银行通知存款 | 0.8-1.35% |
| 短期理财 | 1-90天理财产品 | 1.5-3% |
| 结构性存款 | 保本浮动收益 | 2-4% |
| 国债逆回购 | 交易所逆回购 | 1.5-5% |

## 资金调度方案

### 调度原则

1. **安全性**：资金安全第一
2. **流动性**：保证正常运营所需
3. **收益性**：闲置资金最大化收益

### 调度流程

```
1. 资金预测 → 未来7/30/90天资金需求
2. 闲置识别 → 计算闲置资金量
3. 方案制定 → 投资品种、期限、金额
4. 审批执行 → 按权限审批后执行
5. 到期回收 → 本息回到账户
```

### 审批要求

| 金额 | 审批层级 | 要求 |
|------|----------|------|
| <500万 | 财务负责人 | 日内完成 |
| 500-2000万 | 财务副总 | 2日内完成 |
| >2000万 | 总经理 | 3日内完成 |

## 现金流预测

### 预测周期

| 周期 | 准确度要求 | 更新频率 |
|------|------------|----------|
| 7天预测 | >95% | 每日 |
| 30天预测 | >90% | 每周 |
| 90天预测 | >80% | 每月 |

### 预测模型

```
预计余额 = 当前余额 + 预计收款 - 预计付款 - 已审批支出
```

**预测数据来源**：
- 已签合同付款条款
- 客户付款历史规律
- 银行承兑汇票到期
- 信用证到期

## 风险控制

### 风险类型

| 风险 | 描述 | 控制措施 |
|------|------|----------|
| 流动性风险 | 无法及时支付 | 安全库存、备用授信 |
| 信用风险 | 交易对手违约 | 交易对手准入 |
| 市场风险 | 投资亏损 | 限定投资品种 |
| 操作风险 | 付款错误 | 双人复核、审批流程 |

### 预警机制

- 账户余额 < 安全库存：黄色预警
- 单笔大额支出未审批：红色预警
- 银行授信额度使用 > 80%：橙色预警

## 回复模板

- 资金日报：`💵 资金日报：今日余额{余额}万，较昨日{增减}万，本月累计收支{收}/{支}`
- 闲置提醒：`💡 闲置资金{金额}万可调度，建议{处理方式}，预期收益{收益}`
- 调度执行：`✅ 资金调度完成：转出{账户}，转入{账户}，金额{金额}，期限{期限}`
- 预测报告：`📊 {期限}现金流预测：预计收支{收}/{支}，期末余额{余额}，{充足/不足}`
- 风险预警：`⚠️ 资金风险预警：{内容}，建议{措施}`
