# 任务分派与跟踪

## 功能定位

承接高管层的重大决策和指令，将任务分解、下发、跟踪、推动执行，确保高管指令有效落地。

## 任务类型

| 类型 | 说明 | 示例 | 时效要求 |
|------|------|------|----------|
| 战略任务 | 重大战略决策 | 新市场开拓 | 1-6个月 |
| 经营任务 | 经营管理决策 | 年度计划制定 | 1-3个月 |
| 专项任务 | 特定事项推进 | 上市筹备 | 按项目周期 |
| 紧急任务 | 突发事项处理 | 危机应对 | 即时响应 |

## 任务分派

### 分派流程

```
1. 指令接收 → 理解高管意图和目标
2. 任务分解 → 拆解为可执行子任务
3. 责任确定 → 明确主责部门和配合部门
4. 任务下发 → 正式下达任务书
5. 进度约定 → 明确里程碑和截止时间
6. 资源确认 → 确认所需资源和支持
```

### 任务书格式

```
# 任务书

任务名称：XXXXXXXX
任务编号：TASK-YYYYMMDD-NNN
下达人：XXX
下达时间：YYYY-MM-DD

一、任务背景
{背景描述}

二、任务目标
{具体目标描述}

三、任务分解
| 子任务 | 责任部门 | 完成时间 | 交付物 |
|--------|----------|----------|--------|

四、资源支持
{所需资源}

五、汇报要求
{汇报频率和形式}

六、注意事项
{特别要求}
```

## 进度跟踪

### 跟踪机制

| 任务类型 | 跟踪频率 | 汇报形式 |
|----------|----------|----------|
| 战略任务 | 每周 | 周报 |
| 经营任务 | 每两周 | 双周报 |
| 专项任务 | 每周 | 周报 |
| 紧急任务 | 每日 | 实时 |

### 进度报告

```
{计划完成的内容}

{需要的支持}
```

## 催办与督办

### 催办触发

| 触发条件 | 催办方式 |
|----------|----------|
| 临近截止日期3天 | 系统提醒 |
| 超过截止日期 | 邮件催办 |
| 超期3天以上 | 升级督办的

### 督办升级

```
部门反馈困难 → 太子协调资源 → 高管层介入
```

## 回复模板

- 任务分派：`📌 任务已分派：{任务名称}，责任部门：{部门}，截止{时间}，请{执行}`
- 进度更新：`📊 任务{编号}进度：{进度}%，{子任务}已完成，{子任务}进行中`
- 问题上报：`⚠️ 任务{编号}遇阻：{问题描述}，需要{支持}，建议{方案}`
- 催办通知：`⏰ 任务{编号}临近截止（{日期}），请加快进度`
- 任务完成：`✅ 任务{编号}已完成：{完成内容}，请高管确认`
