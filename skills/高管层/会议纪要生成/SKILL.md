# 会议纪要生成

## 功能定位

记录高管层各类会议内容，整理形成规范会议纪要，明确决议事项和落实责任，跟踪执行情况。

## 会议类型

| 会议类型 | 频率 | 参加人 | 主责 |
|----------|------|--------|------|
| 董事会 | 每季度/临时 | 董事、监事、高管 | 董事会秘书 |
| 经营分析会 | 每月 | 高管、各部门负责人 | 经营管理部 |
| 专题会议 | 按需 | 相关人员 | 召集部门 |
| 高管碰头会 | 每周 | 高管 | 办公室 |

## 纪要生成流程

```
1. 会议记录 → 实时记录或录音转写
2. 初步整理 → 24小时内完成初稿
3. 内容确认 → 主持人确认
4. 正式下发 → 发送全体参会人员
5. 存档归档 → 纸质+电子存档
```

## 决议跟踪

### 决议台账

```
| 编号 | 决议内容 | 责任部门 | 责任人 | 完成时间 | 状态 | 进展 |
|------|----------|----------|--------|----------|------|------|
| 1 | | | | | 进行中 | 已完成60% |
| 2 | | | | | 已完成 | 全部完成 |
```

### 跟踪机制

| 时间节点 | 跟踪动作 |
|----------|----------|
| 完成后3天 | 确认开始执行 |
| 完成前3天 | 确认能否按时完成 |
| 到期日 | 确认完成情况 |
| 超期 | 发出督办通知 |

## 会议类型纪要要点

### 董事会纪要

- 审议议案内容
- 董事意见
- 表决结果（同意/反对/弃权）
- 特别决议（需2/3多数）
- 授权委托情况

### 经营分析会纪要

- 各部门汇报要点
- 重大问题讨论
- KPI完成情况
- 资源协调事项
- 下阶段重点

### 专题会议纪要

- 背景和问题
- 各方观点
- 方案比选
- 决策结论
- 行动方案

## 回复模板

- 纪要已生成：`📝 {会议}纪要已生成，请{主持人}审阅确认`
- 纪要确认：`✅ 纪要已确认，已发送全体参会人员`
- 决议跟踪：`📊 决议执行情况：进行中{进行中}项，已完成{已完成}项，超期{超期}项`
- 督办提醒：`⏰ 决议{编号}将于{日期}到期，责任部门{部门}，请加快进度`
- 决议完成：`✅ 决议{编号}已完成：{完成内容}`
