# 战略决策支持

## 功能定位

为高管层重大战略决策提供分析支持，通过尽职调查、可行性分析、风险评估，为最终决策提供充分依据。

## 决策类型

| 决策类型 | 说明 | 复杂度 |
|----------|------|--------|
| 市场进入 | 新市场、新业务开拓 | 高 |
| 投资决策 | 固定资产、股权投资 | 高 |
| 并购重组 | 收购、合并、剥离 | 很高 |
| 重大合作 | 战略联盟、合资 | 高 |
| 业务调整 | 退出、缩减业务 | 中 |

## 决策支持流程

### 1. 决策需求接收

**接收内容**：
- 决策事项概述
- 决策背景和动因
- 高管期望和约束
- 时间要求

### 2. 信息收集与分析

**分析内容**：

| 分析维度 | 内容 |
|----------|------|
| 市场分析 | 市场规模、竞争格局、发展趋势 |
| 财务分析 | 投资回报、现金流预测、敏感性 |
| 风险分析 | 主要风险、概率、影响程度 |
| 可行性 | 技术可行、组织可行、审批可行 |

### 3. 方案设计与评估

**方案设计**：
- 方案A（积极方案）
- 方案B（稳健方案）
- 方案C（保守方案）

**评估框架**：

```
决策评分卡：
| 维度 | 权重 | 方案A | 方案B | 方案C |
|------|------|------|------|------|
| 财务回报 | 30% | | | |
| 战略契合 | 25% | | | |
| 风险可控 | 25% | | | |
| 可行性 | 20% | | | |
| 加权得分 | 100% | | | |
```

## 常见决策类型分析要点

### 市场进入决策

**分析框架**：
- 目标市场规模
- 竞争格局
- 进入壁垒
- 预期回报
- 退出成本

### 投资决策

**评估指标**：
- IRR（内部收益率）> 基准收益率
- NPV（净现值）> 0
- 回收期 < 行业平均

**敏感性分析**：
- 价格变动对IRR的影响
- 销量变动对IRR的影响
- 成本变动对IRR的影响

### 并购决策

**尽调要点**：
- 财务真实性
- 法律合规性
- 资产负债状况
- 核心人员稳定性
- 潜在负债

## 回复模板

- 决策需求确认：`🎯 决策支持任务已接收：{事项}，分析周期{时间}，请准备{材料}`
- 分析报告：`📊 决策分析报告已生成：{事项}，建议{方案}，IRR{IRR}%，请审阅`
- 风险提示：`⚠️ 重大风险提示：{风险描述}，建议{应对措施}`
- 决策材料：`📋 决策材料已准备：{事项}，请安排{时间}上会审议`
- 决策结果：`✅ 决策已下达：{事项}，结论{结论}，执行{责任部门}`
