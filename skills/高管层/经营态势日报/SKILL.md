# 经营态势日报

## 功能定位

每日汇总集团经营关键信息，为高管提供全面、准确、简洁的经营态势概览，支撑每日快速决策。

## 日报内容

### 板块结构

```
# 经营态势日报

日期：YYYY-MM-DD（星期X）
编报：经营管理部
```

### 一、今日要情

**业务动态**：
- 今日签订重要合同
- 今日重大业务进展
- 重要客户/供应商动态

**市场动态**：
- 今日市场重要价格变化
- 重大行业政策变化
- 竞争对手重要动态

**内部动态**：
- 重要制度/流程更新
- 重大人事变动
- IT系统重要事件

## 生成机制

### 数据来源

| 数据类型 | 来源系统 | 更新时间 |
|----------|----------|----------|
| 贸易数据 | ERP | 每日18:00 |
| 财务数据 | 财务系统 | 每日17:00 |
| 期货数据 | 期货交易系统 | 实时 |
| 汇率数据 | 外汇系统 | 实时 |

### 生成时间

- **定时生成**：每日19:00前完成
- **紧急补报**：重大事件发生时即时编报

## 阅读对象

| 读者 | 关注重点 |
|------|----------|
| 董事长 | 战略风险、重大事项 |
| 总经理 | 整体经营、重点问题 |
| 分管副总 | 所管业务板块 |

## 特色设计

### 颜色标识

| 颜色 | 含义 | 使用场景 |
|------|------|----------|
| 🟢 绿色 | 正常 | 指标正常、风险可控 |
| 🟡 黄色 | 关注 | 偏差>5%、轻度预警 |
| 🟠 橙色 | 预警 | 偏差>15%、需关注 |
| 🔴 红色 | 严重 | 偏差>30%、需立即处理 |

### 重点标注

- **【重要】**：需要高管重点关注
- **【紧急】**：需要立即处理
- **【新增】**：今日新增事项
- **【跟踪】**：持续跟进的未完成事项

## 回复模板

- 日报已生成：`📰 经营态势日报{日期}已生成，今日要情{数量}条，风险提示{数量}条，请查阅`
- 重要事项：`⚠️ 重要事项：{描述}，请{高管}关注`
- 风险预警：`🚨 {类型}预警：{描述}，等级{等级}，建议{措施}`
- 数据更新：`📊 今日核心指标已更新：{指标1}{数值}，{指标2}{数值}`
- 日报补报：`📰 经营态势补充日报：{重大事件描述}，请关注`
